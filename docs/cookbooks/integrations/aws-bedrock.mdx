此示例演示如何在 Python 中配置和使用“mem0ai”开发工具包与 **AWS Bedrock** 和 **OpenSearch Service (AOSS)** 来实现持久内存功能。

## 安装

安装所需的依赖项以包含 Amazon 数据堆栈，包括 **boto3**、**opensearch-py** 和 **langchain-aws**：```bash
pip install "mem0ai[extras]"
```

## 配置与使用

这将 Mem0 设置为：
- [LLM 的 AWS Bedrock](https://docs.mem0.ai/components/llms/models/aws_bedrock)
- [用于嵌入的 AWS Bedrock](https://docs.mem0.ai/components/embedders/models/aws_bedrock#aws-bedrock)
- [OpenSearch 作为矢量存储](https://docs.mem0.ai/components/vectordbs/dbs/opensearch)```python
import boto3
from opensearchpy import RequestsHttpConnection, AWSV4SignerAuth
from mem0 import Memory

region = 'us-west-2'
service = 'aoss'
credentials = boto3.Session().get_credentials()
auth = AWSV4SignerAuth(credentials, region, service)

config = {
    "embedder": {
        "provider": "aws_bedrock",
        "config": {
            "model": "amazon.titan-embed-text-v2:0"
        }
    },
    "llm": {
        "provider": "aws_bedrock",
        "config": {
            "model": "us.anthropic.claude-3-7-sonnet-20250219-v1:0",
            "temperature": 0.1,
            "max_tokens": 2000
        }
    },
    "vector_store": {
        "provider": "opensearch",
        "config": {
            "collection_name": "mem0",
            "host": "your-opensearch-domain.us-west-2.es.amazonaws.com",
            "port": 443,
            "http_auth": auth,
            "connection_class": RequestsHttpConnection,
            "pool_maxsize": 20,
            "use_ssl": True,
            "verify_certs": True,
            "embedding_model_dims": 1024,
        }
    },
}

# Initialize the memory system
m = Memory.from_config(config)
```
## 用法

### 添加内存```python
messages = [
    {"role": "user", "content": "I'm planning to watch a movie tonight. Any recommendations?"},
    {"role": "assistant", "content": "How about thriller movies? They can be quite engaging."},
    {"role": "user", "content": "I'm not a big fan of thriller movies but I love sci-fi movies."},
    {"role": "assistant", "content": "Got it! I'll avoid thriller recommendations and suggest sci-fi movies in the future."}
]

# Store inferred memories (default behavior)
result = m.add(messages, user_id="alice", metadata={"category": "movie_recommendations"})
```
### 搜索记忆```python
relevant_memories = m.search(query, filters={"user_id": "alice"})
```
### 获取所有记忆```python
all_memories = m.get_all(filters={"user_id": "alice"})
```
### 获取特定内存```python
memory = m.get(memory_id)
```
## 结论

借助 Mem0 和 Bedrock 和 OpenSearch 等 AWS 服务，您可以构建智能 AI 伴侣，随着时间的推移，它们会记住、适应和个性化他们的响应。这使得它们成为长期助理、导师或具有持久记忆和自然对话能力的支持机器人的理想选择。
