本指南演示了如何将 OpenAI 的语音应用程序代理 SDK 与 Mem0 的内存功能相结合，创建一个能够记住用户偏好和过去交互的语音助手。

## 先决条件

在开始之前，请确保您拥有：

1. 安装具有语音依赖项的 OpenAI Agents SDK：```bash
pip install 'openai-agents[voice]'
```
2.安装Mem0 SDK：```bash
pip install mem0ai
```
3.安装其他所需的依赖项：```bash
pip install numpy sounddevice pydantic
```
4. 设置您的 API 密钥：
   - Agents SDK 的 OpenAI API 密钥
   - 来自 Mem0 平台的 Mem0 API 密钥

## 代码分解

让我们分解一下这个实现的关键组件：

### 1. 设置依赖项和环境```python
# OpenAI Agents SDK imports
from agents import (
    Agent,

from agents.voice import (

本节处理：
- 从 OpenAI Agents SDK 和 Mem0 导入所需模块
- 设置 API 密钥的环境变量
- 定义一个简单的用户识别系统（使用全局变量）
- 初始化将处理内存操作的 Mem0 客户端

### 2.带有函数装饰器的记忆工具

`@function_tool` 装饰器将 Python 函数转换为 OpenAI 代理的可调用工具。以下是关键的记忆工具：

#### 存储用户记忆```python
import logging

# Set up logging at the top of your file
logging.basicConfig(
    level=logging.DEBUG,
    format='%(asctime)s - %(name)s - %(levelname)s - %(message)s',
    force=True
)
logger = logging.getLogger("memory_voice_agent")

# Then use logger in your function tools
@function_tool
async def save_memories(
    memory: str
) -> str:
    """Store a user memory in memory."""
    # This will be visible in your console
    logger.debug(f"Saving memory: {memory} for user {USER_ID}")
    
    # Store the preference in Mem0
    memory_content = f"User memory - {memory}"
    await mem0_client.add(
        memory_content,
        user_id=USER_ID,
    )

    return f"I've saved your memory: {memory}"
```
这个功能：
- 获取一个内存字符串
- 创建格式化的内存字符串
- 使用`add()`方法将其存储在Mem0中
- 包括元数据来对内存进行分类，以便于检索
- 返回代理将发言的确认消息

#### 寻找相关记忆```python
@function_tool
async def search_memories(
    query: str
) -> str:
    """
    Find memories relevant to the current conversation.
    Args:
        query: The search query to find relevant memories
    """
    print(f"Finding memories related to: {query}")
    results = await mem0_client.search(
        query,
        filters={"user_id": USER_ID},
        top_k=5,
        threshold=0.7,  # Higher threshold for more relevant results
    )

    # Format and return the results
    if not results.get('results', []):
        return "I don't have any relevant memories about this topic."

    memories = [f"• {result['memory']}" for result in results.get('results', [])]
    return "Here's what I remember that might be relevant:\n" + "\n".join(memories)
```
这个工具：
- 获取搜索查询字符串
- 将其传递给Mem0的语义搜索以查找相关记忆
- 设置相关性阈值以确保质量结果
- 返回相关记忆的格式化列表或默认消息

### 3. 创建语音代理```python
def create_memory_voice_agent():
    # Create the agent with memory-enabled tools
    agent = Agent(
        name="Memory Assistant",
        instructions=prompt_with_handoff_instructions(
            """You're speaking to a human, so be polite and concise.
            Always respond in clear, natural English.
            You have the ability to remember information about the user.
            Use the save_memories tool when the user shares an important information worth remembering.
            Use the search_memories tool when you need context from past conversations or user asks you to recall something.
            """,
        ),
        model="gpt-5-mini",
        tools=[save_memories, search_memories],
    )
    
    return agent
```
这个功能：
- 创建具有特定指令的 OpenAI 代理
- 将其配置为使用 gpt-4.1-nano （您可以使用其他型号）
- 向代理注册与内存相关的工具
- 使用“prompt_with_handoff_instructions”来包含标准语音代理行为

### 4. 麦克风录音功能```python
async def record_from_microphone(duration=5, samplerate=24000):
    """Record audio from the microphone for a specified duration."""
    print(f"Recording for {duration} seconds...")
    
    # Create a buffer to store the recorded audio
    frames = []
    
    # Callback function to store audio data
    def callback(indata, frames_count, time_info, status):
        frames.append(indata.copy())
    
    # Start recording
    with sd.InputStream(samplerate=samplerate, channels=1, callback=callback, dtype=np.int16):
        await asyncio.sleep(duration)
    
    # Combine all frames into a single numpy array
    audio_data = np.concatenate(frames)
    return audio_data
```
这个功能：
- 创建简单的异步麦克风录音功能
- 使用声音设备库捕获音频输入
- 录制期间将帧存储在缓冲区中
- 完成后将帧组合成单个 numpy 数组
- 返回音频数据进行处理

### 5. 主循环和语音处理```python
async def main():
    # Create the agent
    agent = create_memory_voice_agent()
    
    # Set up the voice pipeline
    pipeline = VoicePipeline(
        workflow=SingleAgentVoiceWorkflow(agent)
    )
    
    # Configure TTS settings
    pipeline.config.tts_settings.voice = "alloy"
    pipeline.config.tts_settings.speed = 1.0
    
    try:
        while True:
            # Get user input
            print("\nPress Enter to start recording (or 'q' to quit)...")
            user_input = input()
            if user_input.lower() == 'q':
                break
            
            # Record and process audio
            audio_data = await record_from_microphone(duration=5)
            audio_input = AudioInput(buffer=audio_data)
            result = await pipeline.run(audio_input)
            
            # Play response and handle events
            player = sd.OutputStream(samplerate=24000, channels=1, dtype=np.int16)
            player.start()
            
            agent_response = ""
            print("\nAgent response:")
            
            async for event in result.stream():
                if event.type == "voice_stream_event_audio":
                    player.write(event.data)
                elif event.type == "voice_stream_event_content":
                    content = event.data
                    agent_response += content
                    print(content, end="", flush=True)
            
            # Save the agent's response to memory
            if agent_response:
                try:
                    await mem0_client.add(
                        f"Agent response: {agent_response}", 
                        user_id=USER_ID,
                        metadata={"type": "agent_response"}
                    )
                except Exception as e:
                    print(f"Failed to store memory: {e}")
    
    except KeyboardInterrupt:
        print("\nExiting...")
```
这个主函数协调整个过程：
1. 创建具有记忆功能的语音代理
2. 使用 TTS 设置设置语音管道
3. 实现用于录制和处理语音输入的交互循环
4. 处理响应事件流（音频和文本）
5. 自动将座席的响应保存到内存中
6. 包括适当的错误处理和退出机制

## 此实现的主要特点

该实现提供了几个关键特性：

1. **简化用户管理**：为了简单起见，使用全局“USER_ID”变量，但可以扩展以管理多个用户。

2. **真实麦克风输入**：包括“record_from_microphone()”函数，可捕获来自麦克风的实际语音输入。

3. **交互式语音循环**：实现连续的交互循环，允许多次来回交换。

4. **内存管理工具**：
   - `save_memories`：将用户记忆存储在 Mem0 中
   - `search_memories`：搜索相关的过去信息

5. **语音配置**：演示如何配置语音响应的 TTS 设置。

## 运行示例

要运行此示例：

1. 将占位符 API 密钥替换为您的实际密钥
2.确保麦克风连接正确
3.使用 Python 3.8 或更高版本运行脚本
4. 按 Enter 开始录音，然后说出您的请求
5. 按“q”退出应用程序

代理将听取您的请求，通过 OpenAI 模型对其进行处理，根据需要利用 Mem0 进行内存操作，并通过文本输出和语音进行响应。## 带记忆的语音代理的最佳实践

1. **优化语音记忆**：保持记忆简洁且与语音响应相关。

2. **遗忘机制**：实现一种删除或过期不再相关的记忆的方法。

3. **上下文保留**：在每个记忆中存储足够的上下文以使检索有效。

4. **错误处理**：为内存操作实现强大的错误处理，因为即使内存操作失败，语音交互也应该继续顺利进行。

通过将 OpenAI 的 Agents SDK 与 Mem0 的内存功能相结合，您可以创建语音代理，以持久存储用户偏好和过去的交互。这使对话更加自然和个性化，从而显着增强了用户体验。在构建语音应用程序时，请尝试不同的内存策略和过滤方法，以找到针对您的特定用例的全面内存和高效检索之间的最佳平衡。
