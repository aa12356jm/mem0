使用 Mem0 的 YouTube 助手增强您的 YouTube 体验，这是一款 Chrome 扩展程序，可将人工智能支持的聊天直接带入您的 YouTube 视频。无需离开页面，即可利用您自己的知识和记忆获得有关视频内容的即时、个性化答案。

## 特点

- **情境人工智能聊天**：询问有关您正在观看的视频的问题
- **无缝集成**：聊天界面位于 YouTube 的本机 UI 旁边
- **内存集成**：通过 Mem0 根据您的知识进行个性化响应
- **实时记忆**：记忆根据您的互动实时更新

## 安装

Chrome 网上应用店尚未提供此扩展程序。您可以使用以下方法手动安装它：

### 手动安装（开发者模式）1. **下载扩展**：从 [Mem0 GitHub 存储库](https://github.com/mem0ai/mem0/tree/main/examples) 克隆或下载扩展文件
2. **构建**：运行“npm install”，然后运行“npm run build”来安装依赖项并构建扩展
3. **访问 Chrome 扩展**：打开 Google Chrome 并导航至“chrome://extensions”
4. **启用开发者模式**：切换右上角的“开发者模式”开关
5. **加载解压的扩展**：点击“加载解压”，选择包含扩展文件的目录
6. **确认安装**：Mem0 YouTube Assistant 扩展现在应该出现在您的 Chrome 工具栏中

## 设置

1. **配置 API 设置**：单击扩展程序图标并输入您的 OpenAI API 密钥（使用扩展程序所需）
2. **自定义设置**：配置其他设置，例如型号、温度和内存设置
3.**导航到 YouTube**：开始在任何 YouTube 视频上使用助手
4. **记忆**：输入您的 Mem0 API 密钥以启用个性化响应，并从设置中提供初始记忆

- “解释一下他们刚才提到的概念”

## 隐私和数据安全

您的 API 密钥存储在您本地的浏览器中。您的消息将发送到 Mem0 API 以提取和检索内存。 Mem0 致力于确保您数据的隐私和安全。
