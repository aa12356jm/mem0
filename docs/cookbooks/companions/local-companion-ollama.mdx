通过将 Ollama 用于嵌入模型和语言模型 (LLM)，Mem0 可以完全在本地使用。本指南将引导您完成必要的步骤，并提供完整的代码以帮助您入门。

## 概述

通过使用 Ollama，您可以在本地运行 Mem0，这可以更好地控制您的数据和模型。此设置将 Ollama 用于嵌入模型和语言模型，提供完全本地的解决方案。

## 设置

开始之前，请确保您已在本地计算机上安装并正确配置了 Mem0 和 Ollama。

## 完整代码示例

以下是通过 Ollama 在本地设置和使用 Mem0 的完整代码：```python
from mem0 import Memory

config = {
    "vector_store": {
        "provider": "qdrant",
        "config": {
            "collection_name": "test",
            "host": "localhost",
            "port": 6333,
            "embedding_model_dims": 768,  # Change this according to your local model's dimensions
        },
    },
    "llm": {
        "provider": "ollama",
        "config": {
            "model": "llama3.1:latest",
            "temperature": 0,
            "max_tokens": 2000,
            "ollama_base_url": "http://localhost:11434",  # Ensure this URL is correct
        },
    },
    "embedder": {
        "provider": "ollama",
        "config": {
            "model": "nomic-embed-text:latest",
            # Alternatively, you can use "snowflake-arctic-embed:latest"
            "ollama_base_url": "http://localhost:11434",
        },
    },
}

# Initialize Memory with the configuration
m = Memory.from_config(config)

# Add a memory
m.add("I'm visiting Paris", user_id="john")

# Retrieve memories
memories = m.get_all(filters={"user_id": "john"})
```
## 要点

- **配置**：设置涉及配置向量存储、语言模型和嵌入模型以使用本地资源
- **矢量存储**：Qdrant 用作矢量存储，在本地主机上运行
- **语言模型**：Ollama 用作 LLM 提供者，采用 `llama3.1:latest` 模型
- **嵌入模型**：Ollama 也用于嵌入，具有 `nomic-embed-text:latest` 模型

## 结论

使用 Ollama 的 Mem0 本地设置为内存管理和 AI 交互提供了完全独立的解决方案。它允许更好地控制您的数据和模型，同时仍然利用 Mem0 的强大功能。
