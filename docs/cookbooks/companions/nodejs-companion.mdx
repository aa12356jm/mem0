您可以使用 Mem0 创建个性化的 AI Companion。本指南将引导您完成必要的步骤，并提供完整的代码以帮助您入门。

## 概述

个性化 AI Companion 利用 Mem0 在交互过程中保留信息，从而实现量身定制的学习体验。它为每个用户交互创建记忆，并与 OpenAI 的 GPT 模型集成，为用户查询提供详细的上下文感知响应。

## 设置

在开始之前，请确保已安装 Node.js 并创建一个新项目。使用 npm 安装所需的依赖项：```bash
npm install openai mem0ai
```
## 完整代码示例

以下是使用 Mem0 创建 AI Companion 并与之交互的完整代码：```javascript
import { OpenAI } from 'openai';
import { Memory } from 'mem0ai/oss';
import * as readline from 'readline';

const openaiClient = new OpenAI();
const memory = new Memory();

async function chatWithMemories(message, userId = "default_user") {
  const relevantMemories = await memory.search(message, { userId: userId });
  
  const memoriesStr = relevantMemories.results
    .map(entry => `- ${entry.memory}`)
    .join('\n');
  
  const systemPrompt = `You are a helpful AI. Answer the question based on query and memories.
User Memories:
${memoriesStr}`;
  
  const messages = [
    { role: "system", content: systemPrompt },
    { role: "user", content: message }
  ];
  
  const response = await openaiClient.chat.completions.create({
    model: "gpt-5-mini",
    messages: messages
  });
  
  const assistantResponse = response.choices[0].message.content || "";
  
  messages.push({ role: "assistant", content: assistantResponse });
  await memory.add(messages, { userId: userId });
  
  return assistantResponse;
}

async function main() {
  const rl = readline.createInterface({
    input: process.stdin,
    output: process.stdout
  });
  
  console.log("Chat with AI (type 'exit' to quit)");
  
  const askQuestion = () => {
    return new Promise((resolve) => {
      rl.question("You: ", (input) => {
        resolve(input.trim());
      });
    });
  };
  
  try {
    while (true) {
      const userInput = await askQuestion();
      
      if (userInput.toLowerCase() === 'exit') {
        console.log("Goodbye!");
        rl.close();
        break;
      }
      
      const response = await chatWithMemories(userInput, "sample_user");
      console.log(`AI: ${response}`);
    }
  } catch (error) {
    console.error("An error occurred:", error);
    rl.close();
  }
}

main().catch(console.error);
```
### 关键组件

1. **初始化**
   - 该代码初始化 OpenAI 和 Mem0 内存客户端
   - 使用 Node.js 的内置 readline 模块进行命令行交互

2. **内存管理（chatWithMemories功能）**
   - 使用 Mem0 的搜索功能检索相关记忆
   - 构建一个包含过去记忆的系统提示
   - 对 OpenAI 进行 API 调用以生成响应
   - 在内存中存储新的交互

3. **互动聊天界面（主要功能）**
   - 创建用于用户交互的命令行界面
   - 处理用户输入并显示人工智能响应
   - 包括优雅的退出功能

### 环境设置

确保设置环境变量：```bash
export OPENAI_API_KEY=your_api_key
```
### 结论

此实现演示了如何创建一个 AI Companion，使用 Mem0 的内存功能来维护跨对话的上下文。系统自动存储和检索相关信息，创造更加个性化和情境感知的交互体验。

当用户与系统交互时，Mem0 的记忆系统不断学习和适应，使未来的响应更加相关和个性化。这种设置非常适合创建长期学习的人工智能助手，可以维护上下文并随着时间的推移提供越来越个性化的响应。
