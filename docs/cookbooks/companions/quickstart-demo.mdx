## 概述

个性化 AI Companion 利用 Mem0 在交互过程中保留信息，从而实现量身定制的学习体验。它为每个用户交互创建记忆，并与 OpenAI 的 GPT 模型集成，为用户查询提供详细的上下文感知响应。

## 设置

在开始之前，请按照以下步骤设置演示应用程序：

1. 克隆 Mem0 存储库：   ```bash
   git clone https://github.com/mem0ai/mem0.git
   ```
2. 导航到演示应用程序文件夹：   ```bash
   cd mem0/examples/mem0-demo
   ```
3.安装依赖项：   ```bash
   pnpm install
   ```
4. 通过在项目根目录中创建包含以下内容的 `.env` 文件来设置环境变量：   ```bash
   OPENAI_API_KEY=your_openai_api_key
   MEM0_API_KEY=your_mem0_api_key
   ```
您可以通过在 <a href="https://app.mem0.ai/dashboard/api-keys?utm_source=oss&utm_medium=cookbook-companions-quickstart" rel="nofollow">Mem0 API Dashboard</a> 注册来获取“MEM0_API_KEY”。

5.启动开发服务器：   ```bash
   pnpm run dev
   ```
## 增强 Next.js 应用程序

演示运行后，您可以通过修改“mem0-demo”文件夹中的组件来自定义和增强 Next.js 应用程序。考虑：

- 集成其他 API 或第三方服务以扩展功能
