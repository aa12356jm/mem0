## 概述

Mem0 是一个用于 LLM 应用程序的自我改进的内存层，可实现个性化的 AI 体验，从而节省成本并让用户满意。关键字 AI 提供完整的 LLM 可观察性。

将 Mem0 与关键字 AI 相结合可以让您：
1. 为您的 AI 应用程序添加持久内存
2. 跟踪会话之间的交互
3. 使用关键字 AI 可观测性监控内存使用和检索
4. 优化代币使用并降低成本

## 设置和配置

安装必要的库：```bash
pip install mem0ai keywordsai-sdk
```
设置您的环境变量：```python
import os

# Set your API keys
os.environ["MEM0_API_KEY"] = "your-mem0-api-key"
os.environ["KEYWORDSAI_API_KEY"] = "your-keywords-api-key"
os.environ["KEYWORDSAI_BASE_URL"] = "https://api.keywordsai.co/api/"
```
## 基本集成示例

以下是将 Mem0 与关键字 AI 结合使用的简单示例：```python
from mem0 import Memory
import os

# Configuration
api_key = os.getenv("MEM0_API_KEY")
keywordsai_api_key = os.getenv("KEYWORDSAI_API_KEY")
base_url = os.getenv("KEYWORDSAI_BASE_URL") # "https://api.keywordsai.co/api/"

# Set up Mem0 with Keywords AI as the LLM provider
config = {
    "llm": {
        "provider": "openai",
        "config": {
            "model": "gpt-5-mini",
            "temperature": 0.0,
            "api_key": keywordsai_api_key,
            "openai_base_url": base_url,
        },
    }
}

# Initialize Memory
memory = Memory.from_config(config)

# Add a memory
result = memory.add(
    "I like to take long walks on weekends.",
    user_id="alice",
    metadata={"category": "hobbies"},
)

print(result)
```
## 与 OpenAI SDK 的高级集成

对于更高级的用例，您可以通过 OpenAI SDK 将 keywords AI 与 Mem0 集成：```python
from openai import OpenAI
import os
import json

# Initialize client
client = OpenAI(
    api_key=os.environ.get("KEYWORDSAI_API_KEY"),
    base_url=os.environ.get("KEYWORDSAI_BASE_URL"),
)

# Sample conversation messages
messages = [
    {"role": "user", "content": "I'm planning to watch a movie tonight. Any recommendations?"},
    {"role": "assistant", "content": "How about thriller movies? They can be quite engaging."},
    {"role": "user", "content": "I'm not a big fan of thriller movies but I love sci-fi movies."},
    {"role": "assistant", "content": "Got it! I'll avoid thriller recommendations and suggest sci-fi movies in the future."}
]

# Add memory and generate a response
response = client.chat.completions.create(
    model="openai/gpt-4.1-nano",
    messages=messages,
    extra_body={
        "mem0_params": {
            "user_id": "test_user",
            "api_key": os.environ.get("MEM0_API_KEY"),
            "add_memories": {
                "messages": messages,
            },
        }
    },
)

print(json.dumps(response.model_dump(), indent=4))
```
有关此集成的详细信息，请参阅官方[Keywords AI Mem0集成文档](https://docs.keywordsai.co/integration/development-frameworks/mem0)。

## 主要特点

1. **记忆整合**：存储和检索过去交互的相关信息
2. **LLM 可观察性**：使用关键字 AI 跟踪内存使用情况和检索模式
3. **会话持久性**：维护多个用户会话之间的上下文
4. **成本优化**：通过高效的内存检索减少令牌使用
