[**Mem0 Memory**](https://github.com/mem0ai/mem0) 与 [Flowise](https://github.com/FlowiseAI/Flowise) 集成，为您的 AI 聊天流提供持久内存功能。 [Flowise](https://flowiseai.com/) 是一款开源低代码工具，可供开发人员使用拖放界面构建定制的 LLM 编排流程和 AI 代理。

## 概述

1. 为 Flowise 聊天流提供持久内存存储
2. 与现有 Flowise 模板无缝集成
3. 兼容Flowise中的各种LLM节点
4.支持自定义内存配置
5. 易于设置和管理

## 先决条件

在使用 Flowise 设置 Mem0 之前，请确保您拥有：

1. [Flowise已安装](https://github.com/FlowiseAI/Flowise#⚡quick-start) (需要 NodeJS >= 18.15.0):```bash
npm install -g flowise
npx flowise start
```
2. 访问 Flowise UI：http://localhost:3000
3. 基本熟悉[Flowise 的 LLM 编排](https://flowiseai.com/#features) 概念

## 设置和配置

### 1. 设置 Flowise

1. 打开 Flowise 应用程序并创建新画布，或从 Flowise 市场选择模板。
2. 在此示例中，我们使用 **Conversation Chain** 模板。
3. 将默认的 **Buffer Memory** 替换为 **Mem0 Memory**。

![Flowise 内存集成](https://raw.githubusercontent.com/FlowiseAI/FlowiseDocs/main/en/.gitbook/assets/mem0/flowise-flow.png)

### 2. 获取您的 Mem0 API 密钥

1. 导航至 <a href="https://app.mem0.ai/dashboard/api-keys?utm_source=oss&utm_medium=integration-flowise" rel="nofollow">Mem0 API 密钥仪表板</a>。
2. 生成或复制现有的 Mem0 API 密钥。

![Mem0 API 密钥](https://raw.githubusercontent.com/FlowiseAI/FlowiseDocs/main/en/。gitbook/assets/mem0/api-key.png)

### 3.配置Mem0凭证

1. 在 Mem0 凭证部分输入 **Mem0 API 密钥**。
2. 根据需要配置其他设置：```typescript
{
  "apiKey": "m0-xxx",
  "userId": "user-123",  // Optional: Specify user ID
  "projectId": "proj-xxx",  // Optional: Specify project ID
  "orgId": "org-xxx"  // Optional: Specify organization ID
}
```

## 高级配置

### 内存设置

![Mem0 设置](https://raw.githubusercontent.com/FlowiseAI/FlowiseDocs/main/en/.gitbook/assets/mem0/settings.png)

可用设置包括：1. **仅搜索模式**：启用内存检索而不创建新内存
2. **Mem0实体**：配置标识符：
   - `user_id`：每个用户的唯一标识符
   - `run_id`：特定会话会话 ID
   - `app_id`：应用程序标识符
   - `agent_id`：AI代理标识符
3. **项目ID**：将内存分配给特定项目
4. **组织ID**：按组织组织记忆

### 平台配置

<a href="https://app.mem0.ai/dashboard/project-settings?utm_source=oss&utm_medium=integration-flowise" rel="nofollow">Mem0 项目设置</a>中提供了其他设置：

1. **自定义指令**：定义内存提取规则
2. **Expiration Date**：设置自动内存清理期限

![Mem0 项目设置](https://raw.githubusercontent.com/FlowiseAI/FlowiseDocs/main/en/.gitbook/assets/mem0/mem0-settings.png)
