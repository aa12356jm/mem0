Mem0 是一个用于 LLM 应用程序的自我改进的内存层，可实现个性化的 AI 体验，从而节省成本并让用户满意。此扩展允许您使用 Mem0 的智能存储系统存储和检索文本片段。在[Raycast Store](https://www.raycast.com/dev_khant/mem0)中找到Mem0即可使用。

## 开始使用

**获取您的 API 密钥**：您需要 Mem0 API 密钥才能使用此扩展：

a.注册 <a href="https://app.mem0.ai?utm_source=oss&utm_medium=integration-raycast" rel="nofollow">app.mem0.ai</a>

b.导航至您的 API 密钥页面

c.复制您的 API 密钥

d.在扩展首选项中输入此密钥

**基本用法**：

- 存储记忆和文本片段
- 检索上下文感知信息
- 管理持久的用户偏好
- 搜索存储的记忆

## 特点

**记住一切**：永远不会丢失重要信息。存储您的人工智能可以稍后回忆的笔记、偏好和对话。

**智能连接**：自动链接相关主题，帮助您发现有用的连接。

**节省成本**：通过有效检索相关信息而不是重新生成响应，减少人工智能使用的支出。

## 这对您有何帮助**更多个人体验**：您的人工智能会记住您的偏好和过去的对话，使互动感觉更加自然。

**学习你的风格**：适应你的工作方式和你的喜好，随着时间的推移变得更有帮助。

**不再重复**：停止重复解释相同的事情。您的人工智能会记住您的背景和偏好。
