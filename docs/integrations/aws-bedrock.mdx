此集成演示了如何将 **Mem0** 与 **AWS Bedrock** 和 **Amazon OpenSearch Service (AOSS)** 结合使用，以在智能代理中启用持久语义内存。

## 概述

在本指南中，您将：

1. 配置 AWS 凭证以启用 Bedrock 和 OpenSearch 访问
2. Set up the Mem0 SDK to use Bedrock for embeddings and LLM
3. 使用 OpenSearch 作为向量存储来存储和检索内存
4. 使用可扩展的云基础设施构建内存感知应用程序

## 先决条件

- AWS 账户有权访问：
  - Bedrock foundation models (e.g., Titan, Claude)
  - 具有已配置域的 OpenSearch 服务
-Python 3.8+
- 有效的 AWS 凭证（通过环境或 IAM 角色）

## 设置和安装

安装所需的包：```bash
pip install mem0ai boto3 opensearch-py
```
设置环境变量。

使用环境变量、IAM 角色或 AWS CLI 配置您的 AWS 凭证。```python
import os

os.environ['AWS_REGION'] = 'us-west-2'
os.environ['AWS_ACCESS_KEY_ID'] = 'AKIA...'
os.environ['AWS_SECRET_ACCESS_KEY'] = 'AS...'
```
## 初始化 Mem0 集成

导入必要的模块并配置Mem0：```python
import boto3
from opensearchpy import OpenSearch, RequestsHttpConnection, AWSV4SignerAuth
from mem0 import Memory

region = 'us-west-2'
service = 'aoss'
credentials = boto3.Session().get_credentials()
auth = AWSV4SignerAuth(credentials, region, service)

config = {
    "embedder": {
        "provider": "aws_bedrock",
        "config": {
            "model": "amazon.titan-embed-text-v2:0"
        }
    },
    "llm": {
        "provider": "aws_bedrock",
        "config": {
            "model": "anthropic.claude-3-5-haiku-20241022-v1:0",
            "temperature": 0.1,
            "max_tokens": 2000
        }
    },
    "vector_store": {
        "provider": "opensearch",
        "config": {
            "collection_name": "mem0",
            "host": "your-opensearch-domain.us-west-2.es.amazonaws.com",
            "port": 443,
            "http_auth": auth,
            "embedding_model_dims": 1024,
            "connection_class": RequestsHttpConnection,
            "pool_maxsize": 20,
            "use_ssl": True,
            "verify_certs": True
        }
    }
}

# Initialize memory system
m = Memory.from_config(config)
```
## 内存操作

将 Mem0 与 Bedrock 支持的 LLM 和 OpenSearch 存储后端结合使用：```python
# Store conversational context
messages = [
    {"role": "user", "content": "I'm planning to watch a movie tonight. Any recommendations?"},
    {"role": "assistant", "content": "How about a thriller?"},
    {"role": "user", "content": "I prefer sci-fi."},
    {"role": "assistant", "content": "Noted! I'll suggest sci-fi movies next time."}
]

m.add(messages, user_id="alice", metadata={"category": "movie_recommendations"})

# Search for memory
relevant = m.search("What kind of movies does Alice like?", filters={"user_id": "alice"})

# Retrieve all user memories
all_memories = m.get_all(filters={"user_id": "alice"})
```
## 主要特点

1. **无服务器内存嵌入**：使用 Titan 或其他 Bedrock 模型实现快速的云原生嵌入
2. **可扩展矢量搜索**：通过 OpenSearch 存储和检索矢量化内存
3. **无缝AWS身份验证**：使用AWS IAM或环境变量进行安全身份验证
4. **用户特定的内存空间**：每个用户 ID 的内存都是隔离的
5. **持久记忆上下文**：跨会话维护和调用历史记录
