[**Mem0 AI SDK Provider**](https://www.npmjs.com/package/@mem0/vercel-ai-provider) 是由 **Mem0** 开发的用于与 Vercel AI SDK 集成的库。该库通过引入持久内存功能，为您的应用程序带来增强的 AI 交互功能。

<注释类型=“信息”>
  Mem0 AI SDK 现在支持 <strong>Vercel AI SDK V5</strong>。
</注>

## 概述

1. 为对话式人工智能提供持久内存存储
2. 与Vercel AI SDK平滑集成
3. 确保与多个LLM提供商的兼容性
4. 支持结构化消息格式，清晰可见
5. 促进流响应能力

## 设置和配置

使用 npm 安装 SDK 提供程序：```bash
npm install @mem0/vercel-ai-provider
```
## 开始使用

### 设置 Mem0

1. 从 <a href="https://app.mem0.ai/dashboard/api-keys?utm_source=oss&utm_medium=integration-vercel-ai-sdk" rel="nofollow">Mem0 仪表板</a>获取您的 **Mem0 API 密钥**。

2. 在您的应用程序中初始化 Mem0 客户端：    ```typescript
    import { createMem0 } from "@mem0/vercel-ai-provider";

    const mem0 = createMem0({
      provider: "openai",
      mem0ApiKey: "m0-xxx",
      apiKey: "provider-api-key",
      config: {
        // Options for LLM Provider
      },
      // Optional Mem0 Global Config
      mem0Config: {
        user_id: "mem0-user-id",
      },
    });
    ```
> **注意**：`openai` 提供程序设置为默认值。考虑使用“MEM0_API_KEY”和“OPENAI_API_KEY”作为安全环境变量。

    > **注意**： `mem0Config` 是可选的。它用于设置 Mem0 客户端的全局配置（例如“user_id”、“agent_id”、“app_id”、“run_id”等）。

3. 添加记忆以增强背景：    ```typescript
    import { LanguageModelV2Prompt } from "@ai-sdk/provider";
    import { addMemories } from "@mem0/vercel-ai-provider";

    const messages: LanguageModelV2Prompt = [
      { role: "user", content: [{ type: "text", text: "I love red cars." }] },
    ];

    await addMemories(messages, { user_id: "borat" });
    ```
### 独立功能：    ```typescript
    await addMemories(messages, { user_id: "borat", mem0ApiKey: "m0-xxx" });
    await retrieveMemories(prompt, { user_id: "borat", mem0ApiKey: "m0-xxx" });
    await getMemories(prompt, { user_id: "borat", mem0ApiKey: "m0-xxx" });
    ```
> 对于独立功能，例如“addMemories”、“retrieveMemories”和“getMemories”，您必须将“MEM0_API_KEY”设置为环境变量或直接在函数调用中传递它。

     > `getMemories` 将以对象数组的形式返回原始记忆，而 `retrieveMemories` 将以字符串格式返回响应，并显示包含检索到的记忆的系统提示符。

     > `getMemories` 返回内存对象数组。

### 1. 使用内存上下文生成基本文本    ```typescript
    import { generateText } from "ai";
    import { createMem0 } from "@mem0/vercel-ai-provider";

    const mem0 = createMem0();

    const { text } = await generateText({
      model: mem0("gpt-4-turbo", { user_id: "borat" }),
      prompt: "Suggest me a good car to buy!",
    });
    ```
### 2. 将 OpenAI Provider 与 Memory Utils 相结合    ```typescript
    import { generateText } from "ai";
    import { openai } from "@ai-sdk/openai";
    import { retrieveMemories } from "@mem0/vercel-ai-provider";

    const prompt = "Suggest me a good car to buy.";
    const memories = await retrieveMemories(prompt, { user_id: "borat" });

    const { text } = await generateText({
      model: openai("gpt-4-turbo"),
      prompt: prompt,
      system: memories,
    });
    ```

### 3. 带有内存上下文的流式响应    ```typescript
    import { streamText } from "ai";
    import { createMem0 } from "@mem0/vercel-ai-provider";

    const mem0 = createMem0();

    const { textStream } = streamText({
        model: mem0("gpt-4-turbo", {
            user_id: "borat",
        }),
        prompt: "Suggest me a good car to buy! Why is it better than the other cars for me? Give options for every price range.",
    });

    for await (const textPart of textStream) {
        process.stdout.write(textPart);
    }
    ```

### 6. 带内存上下文的文件支持

Mem0 AI SDK 支持使用内存上下文进行文件处理。以下是分析 PDF 文件的示例：```typescript
import { streamText } from "ai";
import { createMem0 } from "@mem0/vercel-ai-provider";
import { readFileSync } from 'fs';
import { join } from 'path';

const mem0 = createMem0({
  provider: "google",
  mem0ApiKey: "m0-xxx",
  config: {
    apiKey: "google-api-key"
  },
  mem0Config: {
    user_id: "alice",
  },
});

async function main() {
  // Read the PDF file
  const filePath = join(process.cwd(), 'my_pdf.pdf');
  const fileBuffer = readFileSync(filePath);

  // Convert the file's arrayBuffer to a Base64 data URL
  const arrayBuffer = fileBuffer.buffer.slice(fileBuffer.byteOffset, fileBuffer.byteOffset + fileBuffer.byteLength);
  const uint8Array = new Uint8Array(arrayBuffer);

  // Convert Uint8Array to an array of characters
  const charArray = Array.from(uint8Array, byte => String.fromCharCode(byte));
  const binaryString = charArray.join('');
  const base64Data = Buffer.from(binaryString, 'binary').toString('base64');
  const fileDataUrl = `data:application/pdf;base64,${base64Data}`;

  const { textStream } = streamText({
    model: mem0("gemini-2.5-flash"),
    messages: [
      {
        role: 'user',
        content: [
          {
            type: 'text',
            text: 'Analyze the following PDF and generate a summary.',
          },
          {
            type: 'file',
            data: fileDataUrl,
            mediaType: 'application/pdf',
          },
        ],
      },
    ],
  });

  for await (const textPart of textStream) {
    process.stdout.write(textPart);
  }
}

main();
```
> **注意**：支持多模式功能（例如 Google 的 Gemini 模型）的提供商提供文件支持。该示例展示了如何处理 PDF 文件，但您也可以处理图像、文本文件和其他支持的格式。

## 主要特点

- `createMem0()`：初始化一个新的 Mem0 提供程序实例。
- `retrieveMemories()`：检索提示的内存上下文。
- `getMemories()`：以数组格式从您的个人资料中获取记忆。
- `addMemories()`：添加用户记忆以增强上下文响应。

## 最佳实践

1.**用户标识**：使用唯一的“user_id”来实现一致的内存检索。
2. **内存清理**：定期清理未使用的内存数据。

    > **注意**：我们还支持 `agent_id`、`app_id` 和 `run_id`。请参阅[文档](/api-reference/memory/add-memories)。
