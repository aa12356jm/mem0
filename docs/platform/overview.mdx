# Mem0 平台概述

Mem0 是保持对话上下文的内存引擎，因此用户永远不会重复自己的内容，并且您的代理可以连续响应。 Mem0 平台作为完全托管的服务提供这种体验 - 扩展、保护和丰富内存，而无需您进行任何基础设施工作。

## 为什么这很重要

- **个性化回复**：记忆在用户和座席之间持久存在，从而减少快速膨胀和重复问题。
- **托管堆栈**：Mem0 运行矢量存储、图形服务和重新排序器 - 无需配置、调整或维护。
- **企业控制**：默认情况下提供 SOC 2、审核日志和工作区治理，以做好生产准备。
