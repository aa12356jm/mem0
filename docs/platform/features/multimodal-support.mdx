Mem0 通过支持多模式数据（包括图像和文档），将其功能扩展到文本之外。借助此功能，用户可以将视觉和文档内容无缝集成到他们的交互中，从而使Mem0能够从各种媒体类型中提取相关信息并丰富记忆系统。

## 它是如何工作的

当用户提交图像或文档时，Mem0 对其进行处理以提取文本信息和其他相关详细信息。然后，这些细节将被添加到用户的记忆中，从而增强系统理解和回忆多模式输入的能力。

## 支持的媒体类型

Mem0 目前支持以下媒体类型：

1. **图片** - JPG、PNG等常见图片格式
2. **文档** - MDX、TXT 和 PDF 文件

## 整合方法

### 1. 图片

#### 使用图像 URL

您可以通过提供图像的直接 URL 来包含图像。该方法对于在线图像简单高效。```python {2, 5-13}
# Define the image URL
image_url = "https://www.superhealthykids.com/wp-content/uploads/2021/10/best-veggie-pizza-featured-image-square-2.jpg"

# Create the message dictionary with the image URL
image_message = {
    "role": "user",
    "content": {
        "type": "image_url",
        "image_url": {
            "url": image_url
        }
    }
}
client.add([image_message], user_id="alice")
```
#### 对本地文件使用 Base64 图像编码

对于本地图像或直接嵌入图像更合适时，您可以使用 Base64 编码的字符串。

<代码组>```python Python
import base64

# Path to the image file
image_path = "path/to/your/image.jpg"

# Encode the image in Base64
with open(image_path, "rb") as image_file:
    base64_image = base64.b64encode(image_file.read()).decode("utf-8")

# Create the message dictionary with the Base64-encoded image
image_message = {
    "role": "user",
    "content": {
        "type": "image_url",
        "image_url": {
            "url": f"data:image/jpeg;base64,{base64_image}"
        }
    }
}
client.add([image_message], user_id="alice")
```

```typescript TypeScript
import MemoryClient from "mem0ai";
import fs from 'fs';

const imagePath = 'path/to/your/image.jpg';

const base64Image = fs.readFileSync(imagePath, { encoding: 'base64' });

const imageMessage = {
    role: "user",
    content: {
        type: "image_url",
        image_url: {
            url: `data:image/jpeg;base64,${base64Image}`
        }
    }
};

await client.add([imageMessage], { userId: "alice" })
```
</代码组>

### 2. 文本文档 (MDX/TXT)

Mem0 支持 MDX 或 TXT 格式的在线和本地文本文档。

#### 使用文档 URL```python
# Define the document URL
document_url = "https://www.w3.org/TR/2003/REC-PNG-20031110/iso_8859-1.txt"

# Create the message dictionary with the document URL
document_message = {
    "role": "user",
    "content": {
        "type": "mdx_url",
        "mdx_url": {
            "url": document_url
        }
    }
}
client.add([document_message], user_id="alice")
```
#### 对本地文档使用 Base64 编码```python
import base64

# Path to the document file
document_path = "path/to/your/document.txt"

# Function to convert file to Base64
def file_to_base64(file_path):
    with open(file_path, "rb") as file:
        return base64.b64encode(file.read()).decode('utf-8')

# Encode the document in Base64
base64_document = file_to_base64(document_path)

# Create the message dictionary with the Base64-encoded document
document_message = {
    "role": "user",
    "content": {
        "type": "mdx_url",
        "mdx_url": {
            "url": base64_document
        }
    }
}
client.add([document_message], user_id="alice")
```

## 具有多种文件类型的完整示例

下面是一个全面的示例，展示了如何处理不同的文件类型：```python
import base64
from mem0 import MemoryClient

client = MemoryClient()

def file_to_base64(file_path):
    with open(file_path, "rb") as file:
        return base64.b64encode(file.read()).decode('utf-8')

# Example 1: Using an image URL
image_message = {
    "role": "user",
    "content": {
        "type": "image_url",
        "image_url": {
            "url": "https://example.com/sample-image.jpg"
        }
    }
}

# Example 2: Using a text document URL
text_message = {
    "role": "user",
    "content": {
        "type": "mdx_url",
        "mdx_url": {
            "url": "https://www.w3.org/TR/2003/REC-PNG-20031110/iso_8859-1.txt"
        }
    }
}

# Example 3: Using a PDF URL
pdf_message = {
    "role": "user",
    "content": {
        "type": "pdf_url",
        "pdf_url": {
            "url": "https://www.w3.org/WAI/ER/tests/xhtml/testfiles/resources/pdf/dummy.pdf"
        }
    }
}

# Add each message to the memory system
client.add([image_message], user_id="alice")
client.add([text_message], user_id="alice")
client.add([pdf_message], user_id="alice")
```
使用这些方法，您可以将各种媒体类型无缝地融入您的交互中，进一步增强 Mem0 的多模式功能。
