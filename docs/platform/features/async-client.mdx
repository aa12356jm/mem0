`AsyncMemoryClient` 是一个用于与 Mem0 API 交互的异步客户端。它提供与同步“MemoryClient”类似的功能，但允许非阻塞操作，这在需要高并发性的应用程序中非常有用。

## 初始化

要使用异步客户端，首先需要初始化它：

<代码组>```python Python
import os
from mem0 import AsyncMemoryClient

os.environ["MEM0_API_KEY"] = "your-api-key"

client = AsyncMemoryClient()
```

```javascript JavaScript
const { MemoryClient } = require('mem0ai');
const client = new MemoryClient({ apiKey: 'your-api-key'});
```
</代码组>

## 方法

`AsyncMemoryClient` 提供以下方法：

### 添加

异步添加新内存。

<代码组>```python Python
messages = [
    {"role": "user", "content": "Alice loves playing badminton"},
    {"role": "assistant", "content": "That's great! Alice is a fitness freak"},
]
await client.add(messages, user_id="alice")
```

```javascript JavaScript
const messages = [
    {"role": "user", "content": "Alice loves playing badminton"},
    {"role": "assistant", "content": "That's great! Alice is a fitness freak"},
];
await client.add(messages, { userId: "alice" });
```
</代码组>

### 搜索

根据异步查询搜索内存。

<代码组>```python Python
await client.search("What is Alice's favorite sport?", filters={"user_id": "alice"})
```

```javascript JavaScript
await client.search("What is Alice's favorite sport?", { filters: { userId: "alice" } });
```
</代码组>

### 获取全部

异步检索用户的所有记忆。

<Callout type="warning" title="需要过滤器">
`get_all()` 现在需要指定过滤器。
</标注>

<代码组>```python Python
await client.get_all(filters={"AND": [{"user_id": "alice"}]})
```

```javascript JavaScript
await client.getAll({ filters: {"AND": [{"user_id": "alice"}]} });
```
</代码组>

### 删除

异步删除特定内存。

<代码组>```python Python
await client.delete(memory_id="memory-id-here")
```

```javascript JavaScript
await client.delete("memory-id-here");
```
</代码组>

### 全部删除

异步删除用户的所有记忆。

<代码组>```python Python
await client.delete_all(user_id="alice")
```

```javascript JavaScript
await client.deleteAll({ userId: "alice" });
```
</代码组>

<注意>
  至少需要一个过滤器（“user_id”、“agent_id”、“app_id”或“run_id”）——在没有过滤器的情况下调用“delete_all”会引发错误，以防止意外数据丢失。您可以传递 `"*"` 作为值来删除给定实体类型的所有记忆（例如，`user_id="*"` 删除每个用户的记忆）。完整的项目擦除需要将所有四个过滤器设置为“*”。
</注>

### 历史

异步获取特定内存的历史记录。

<代码组>```python Python
await client.history(memory_id="memory-id-here")
```

```javascript JavaScript
await client.history("memory-id-here");
```
</代码组>

### 用户

异步获取具有与其关联的内存的所有用户、代理和运行。

<代码组>```python Python
await client.users()
```

```javascript JavaScript
await client.users();
```
</代码组>

### 重置

重置客户端，异步删除所有用户和内存。

<代码组>```python Python
await client.reset()
```

```javascript JavaScript
await client.reset();
```
</代码组>

## 结论

`AsyncMemoryClient` 提供了一种与 Mem0 API 异步交互的强大方法，从而实现更高效、响应更快的应用程序。通过使用此客户端，您可以执行内存操作，而不会阻止应用程序的执行。
