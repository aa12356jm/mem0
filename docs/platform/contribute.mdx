# 以正确的方式构建 Mem0 文档

<信息>
  **这是给谁的**
  - 贡献者和法学硕士助理更新文档
  - 审稿人在出版前审查新页面
  - 维护人员将实时文档与模板库同步
</信息>

<步骤>
<Step title="查看标准">
检查您团队的最新清单或指南，以便更新保持正确的导航流程、CTA 模式和语言覆盖范围。
</步骤>
<Step title="选择正确的模板">
选择您正在编写的文档类型（快速入门、功能指南、迁移等）并从下面的模板库复制框架。
</步骤>
<Step title="起草、验证并记录后续内容">
完整填写框架，包括内联验证标注，并在打开 PR 之前记下维护人员的任何未解决问题。</步骤>
</步骤>

<信息图标=“检查”>
  在本地预览时，确认页面以两张 CTA 卡结尾，包括 Python 和 TypeScript 示例（如果存在），并保留所有 Mintlify 图标（无表情符号）。
</信息>
