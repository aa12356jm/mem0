<更新标签=“2026-04-14”描述=“Mem0 SDK v2.0.0 / v3.0.0”>

**新的内存算法 - 最先进的精度，成本降低约 3-4 倍**

内存管道的彻底重写，基准测试改进了 20 多个点：

- **LoCoMo：** 71.4 → **91.6** (+20) — 多轮对话回忆
- **LongMemEval:** 67.8 → **93.4** (+26) — 跨会话的长期记忆
- **BEAM（1M 代币）：** **64.1** — 生产规模内存评估
- **代理记忆是一流的** — 之前的算法：助理回忆率为 46%。新：**100%**
- **时间推理有效**——“在 SF 之前我住在哪里？”上一个：51%。新：**93%**
- **~减少 3-4 倍的标记** — 每次检索少于 7K 个标记，而全上下文方法则需要超过 25K 个标记
- **仅添加提取** - 记忆积累；没有任何内容被覆盖或删除
- **混合检索** — 语义 + BM25 关键字 + 实体提升，并行评分
- **实体链接** — 跨内存提取、嵌入和链接的实体

重大更改：从 OSS 中删除了图形内存，更改了“search()”默认值，删除了已弃用的参数。请参阅[迁移指南](/migration/oss-v2-to-v3)。

</更新>

<更新标签=“2026-04-06”描述=“Mem0技能图”>

**Mem0 技能图 — AI 代理的上下文文档**

Claude Code、Cursor 和 Codex 中的 AI 编码代理现在可以直接在其工作流程中访问 Mem0 知识，无需进行文档搜索。推出了三种相互关联的技能：

- **mem0 核心技能** — 完整的 Python 和 TypeScript SDK 参考、REST API 模式以及 LangChain、CrewAI、Autogen 等的集成指南
- **mem0-cli 技能** — 终端命令参考、配置演练和 CI/CD 配方
- **mem0-vercel-ai-sdk 技能** — Vercel AI SDK 提供商 API、内存增强生成模式和多提供商设置

</更新>

<更新标签=“2026-04-06”描述=“Mem0 CLI v0.2.2”>

**官方 Mem0 CLI — 现在在 PyPI 和 npm 上**Mem0 的全功能命令行界面，可在 Python 和 Node.js 中使用：- **安装：** `pip install mem0-cli` 或 `npm install -g @mem0/cli`
- **完整命令套件** — `add`、`search`、`list`、`get`、`update`、`delete`、`import`、`config`、`init`、`status`、`entity`、`event`
- **交互式设置** - `mem0 init` 带有电子邮件验证或直接 API 密钥输入
- **随处可用** — 平台（Mem0 Cloud）和自托管 OSS 模式
- **可编写脚本** — 用于 CI/CD 管道和自动化的 `--json` 标志
- **双 SDK** — 相同的命令，相同的 Python 和 Node.js 体验

</更新>

<更新标签=“2026-04-06”描述=“OpenClaw v1.0.4”>

**OpenClaw 插件 — 生产就绪**

OpenClaw Mem0 插件从最初发布到生产就绪仅用了一周时间（v1.0.0 → v1.0.4）：

- **基于技能的内存架构** - 具有技能加载器、批量提取和域感知内存分类的新提取管道
- **梦想之门** — 在空闲期间自动进行内存整合，以实现更高质量的长期回忆
- **交互式 CLI** — `openclaw mem0 init`、`status`、`config`、`import` 和 `event` 命令
- **统一工具命名** — `memory_add` 和 `memory_delete` 替换 4 个旧版工具，匹配平台 API
- **安全性强化** — 路径遍历保护、固定依赖项、跨 10 个文件进行 329 项测试

</更新>

在三个主要 AI 开发环境中推出了统一的 Mem0 插件 - 首先是 Claude Code 和 Cursor（3 月 25 日），然后是 Codex（4 月 2 日）：- **9 MCP 内存工具** — 通过“mcp.mem0.ai”添加、搜索、获取、更新、删除、批量删除、实体管理

- **可流式 HTTP 传输** — 用于实时流式传输的新 MCP 传输协议
- **Codex 特定技能** — Codex 工作流程的“mem0-plugin/skills/mem0-codex”专用技能
