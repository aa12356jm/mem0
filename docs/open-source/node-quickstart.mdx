只需几个步骤即可使用 Node SDK 启动 Mem0。您将安装该软件包、初始化客户端、添加内存并通过一次搜索确认检索。

## 先决条件

- Node.js 18 或更高版本
- （可选）当您想要自定义提供程序时，OpenAI API 密钥存储在您的环境中

## 安装并运行你的第一个内存

<步骤>
<步骤标题=“安装SDK”>```bash
npm install mem0ai
```
</步骤>

<Step title="初始化客户端">```ts
import { Memory } from "mem0ai/oss";

const memory = new Memory();
```
</步骤>

<Step title="添加内存">```ts
const messages = [
  { role: "user", content: "I'm planning to watch a movie tonight. Any recommendations?" },
  { role: "assistant", content: "How about thriller movies? They can be quite engaging." },
  { role: "user", content: "I'm not a big fan of thriller movies but I love sci-fi movies." },
  { role: "assistant", content: "Got it! I'll avoid thriller recommendations and suggest sci-fi movies in the future." }
];

await memory.add(messages, { userId: "alice", metadata: { category: "movie_recommendations" } });
```
</步骤>

<步骤标题=“搜索记忆”>```ts
const results = await memory.search("What do you know about me?", { filters: { userId: "alice" } });
console.log(results);
```
**输出**```json
{
  "results": [
    {
      "id": "892db2ae-06d9-49e5-8b3e-585ef9b85b8e",
      "memory": "User is planning to watch a movie tonight.",
      "score": 0.38920719231944799,
      "metadata": {
        "category": "movie_recommendations"
      },
      "userId": "alice"
    }
  ]
}
```
</步骤>
</步骤>

<注意>
默认情况下，Node SDK 使用本地友好设置（OpenAI `gpt-5-mini`、`text-embedding-3-small`、内存向量存储和 SQLite 历史记录）。通过传递配置来交换组件，如下所示。
</注>

## 配置生产```ts
import { Memory } from "mem0ai/oss";

const memory = new Memory({
  embedder: {
    provider: "openai",
    config: {
      apiKey: process.env.OPENAI_API_KEY || "",
      model: "text-embedding-3-small"
    }
  },
  vectorStore: {
    provider: "memory",
    config: {
      collectionName: "memories",
      dimension: 1536
    }
  },
  llm: {
    provider: "openai",
    config: {
      apiKey: process.env.OPENAI_API_KEY || "",
      model: "gpt-4-turbo-preview"
    }
  },
  historyDbPath: "memory.db"
});
```
## 管理记忆（可选）

<代码组>```ts Get all memories
const allMemories = await memory.getAll({ filters: { userId: "alice" } });
console.log(allMemories);
```

```ts Get one memory
const singleMemory = await memory.get("892db2ae-06d9-49e5-8b3e-585ef9b85b8e");
console.log(singleMemory);
```

```ts Search memories
const result = await memory.search("What do you know about me?", { filters: { userId: "alice" } });
console.log(result);
```

```ts Update a memory
const updateResult = await memory.update(
  "892db2ae-06d9-49e5-8b3e-585ef9b85b8e",
  "I love India, it is my favorite country."
);
console.log(updateResult);
```
</代码组>```ts
// Audit history
const history = await memory.history("892db2ae-06d9-49e5-8b3e-585ef9b85b8e");
console.log(history);

// Delete specific or scoped memories
await memory.delete("892db2ae-06d9-49e5-8b3e-585ef9b85b8e");
await memory.deleteAll({ userId: "alice" });

// Reset everything
await memory.reset();
```
## 使用自定义历史记录存储

当您需要无服务器友好的历史存储时，Node SDK 支持 Supabase（或其他提供商）。

<代码组>```ts Supabase provider
import { Memory } from "mem0ai/oss";

const memory = new Memory({
  historyStore: {
    provider: "supabase",
    config: {
      supabaseUrl: process.env.SUPABASE_URL || "",
      supabaseKey: process.env.SUPABASE_KEY || "",
      tableName: "memory_history"
    }
  }
});
```

```ts Disable history
import { Memory } from "mem0ai/oss";

const memory = new Memory({
  disableHistory: true
});
```
</代码组>

使用以下命令创建 Supabase 表：```sql
create table memory_history (
  id text primary key,
  memory_id text not null,
  previous_value text,
  new_value text,
  action text not null,
  created_at timestamp with time zone default timezone('utc', now()),
  updated_at timestamp with time zone,
  is_deleted integer default 0
);
```
## 配置参数

Mem0 提供跨向量存储、LLM、嵌入器和历史存储的精细配置。<手风琴组>
  <手风琴标题=“矢量商店”>

| ---| ---| ---|
| `提供者` |矢量存储提供者（例如“内存”）| `“记忆”` |
| `主机` |主机地址 | `"本地主机"` |
| `端口` |端口号 | `未定义` |
  </手风琴>
  <手风琴标题=“法学硕士”>
|参数|描述 |供应商|

| `温度` |温度值|全部 |
| `apiKey` | API 密钥 |全部 |
| `maxTokens` |生成的最大代币 |全部 |
| `顶P` |概率阈值|全部 |
| `topK` |保持的令牌数量 |全部 |
| `openaiBaseUrl` |基本 URL 覆盖 |开放人工智能 |
