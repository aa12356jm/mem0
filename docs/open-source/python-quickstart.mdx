5 分钟内即可开始使用 Mem0 的 Python SDK。本指南向您展示如何安装 Mem0 并存储您的第一个内存。

## 先决条件

- Python 3.10 或更高版本
- OpenAI API 密钥（[在此处获取一个](https://platform.openai.com/api-keys)）

设置您的 OpenAI API 密钥：```bash
export OPENAI_API_KEY="your-openai-api-key"
```
<注意>
默认使用 OpenAI。想要使用 Ollama、Anthropic 或本地模型？请参阅[配置](/open-source/configuration)。
</注>

## 安装

<步骤>
<Step title="通过 pip 安装">```bash
pip install mem0ai
```
</步骤>

<步骤标题=“初始化内存”>```python
from mem0 import Memory

m = Memory()

````
</步骤>

<Step title="添加内存">```python
messages = [
    {"role": "user", "content": "Hi, I'm Alex. I love basketball and gaming."},
    {"role": "assistant", "content": "Hey Alex! I'll remember your interests."}
]
m.add(messages, user_id="alex")
````
</步骤>

<步骤标题=“搜索记忆”>```python
results = m.search("What do you know about me?", filters={"user_id": "alex"})
print(results)
```
**输出：**```json
{
  "results": [
    {
      "id": "mem_123abc",
      "memory": "Name is Alex. Enjoys basketball and gaming.",
      "user_id": "alex",
      "categories": ["personal_info"],
      "created_at": "2025-10-22T04:40:22.864647-07:00",
      "score": 0.89
    }
  ]
}
```
</步骤>
</步骤>


<注意>
默认情况下 `Memory()` 连接起来：
- OpenAI `gpt-5-mini` 用于事实提取和更新
- OpenAI `text-embedding-3-small` 嵌入（1536 维）
- Qdrant向量存储在`/tmp/qdrant`上的磁盘数据
- SQLite 历史记录位于 `~/.mem0/history.db`
- 没有重新排名器（需要时在配置中添加一个）
</注>
