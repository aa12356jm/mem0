# 自托管功能概述

Mem0 开源附带了针对生产工作负载调整内存行为的功能——异步操作、图形关系、多模式输入和微调检索。使用代码或 YAML 配置这些功能以满足您的应用程序的需求。

<信息>
  从<Link href="/open-source/python-quickstart">Python 快速入门</Link>开始验证基本内存操作，然后在需要时启用以下功能。
</信息>
