## Mem0 REST API

Mem0 提供全面的 REST API，用于将高级内存功能集成到您的应用程序中。使用简单的 HTTP 请求跨用户、代理和自定义实体创建、搜索、更新和管理内存。

<信息>
**快速开始：** 从 <a href="https://app.mem0.ai/dashboard/api-keys?utm_source=oss&utm_medium=api-reference" rel="nofollow">Mem0 Dashboard</a> 获取 API 密钥，并在几分钟内进行首次内存操作。
</信息>

---

## 快速入门指南

只需三个简单步骤即可开始使用 Mem0 API：

1. **[Add Memories](/api-reference/memory/add-memories)** - 存储用户对话中的信息和上下文
2. **[Search Memories](/api-reference/memory/search-memories)** - 使用语义搜索检索相关记忆
3.**[获取记忆](/api-reference/memory/get-memories)** - 获取特定实体的所有记忆

## 验证

所有 API 请求都需要使用基于令牌的身份验证进行身份验证。在授权标头中包含您的 API 密钥：```bash
Authorization: Token <your-api-key>```
从 <a href="https://app.mem0.ai/dashboard/api-keys?utm_source=oss&utm_medium=api-reference" rel="nofollow">Mem0 仪表板</a>获取 API 密钥。

<警告>
**确保您的 API 密钥安全。**切勿将其暴露在客户端代码或公共存储库中。仅使用环境变量和服务器端请求。
</警告>
