# 文档贡献

## 先决条件

在开始之前，请确保您的系统上安装了 **Node.js（版本 23.6.0 或更高版本）**。

---

## 设置 Minlify

### 第 1 步：安装 Mintlify

使用您首选的包管理器全局安装 Mintlify：

<代码组>```bash npm
npm i -g mintlify
```

```bash yarn
yarn global add mintlify
```
</代码组>

### 步骤 2：运行文档服务器

导航到“docs/”目录（“docs.json”所在的位置）并启动开发服务器：```bash
mintlify dev
```
该文档网站位于：[http://localhost:3000](http://localhost:3000)。

---

## 自定义端口

默认情况下，Mintlify 在 **端口 3000** 上运行。要使用不同的端口，请添加 `--port` 标志：```bash
mintlify dev --port 3333
```
---

通过执行这些步骤，您可以有效地为 Mem0 的文档做出贡献。
