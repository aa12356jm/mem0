# 安全移除记忆

删除内存是您满足合规性请求、撤消不良数据或清理过期会话的方式。 Mem0 可让您删除特定内存、ID 列表或与过滤器匹配的所有内容。

<信息>
  **为什么重要**
  - 满足用户擦除 (GDPR/CCPA) 的要求，而无需触及您的其余数据。
  - 通过删除陈旧或不正确的事实来保持知识库的准确性。
  - 适用于托管平台 API 和 OSS SDK。
</信息>

## 关键术语

- **memory_id** – 由 `add`/`search` 返回的唯一 ID，标识要删除的记录。
- **batch_delete** – API 调用可在一次请求中删除最多 1000 个内存。
- **delete_all** – 按用户、代理、运行或元数据进行基于过滤器的删除。
- **不可变** – 无法更新的标记记忆；而是删除+重新添加。

## 删除流程如何工作<步骤>
<Step title="选择范围">
决定是要删除单个内存、列表还是与过滤器匹配的所有内容。
</步骤>
<Step title="提交删除通话">
使用所需的 ID 或过滤器调用“delete”、“batch_delete”或“delete_all”。
</步骤>
<步骤标题=“验证”>
确认响应消息，然后重新运行“搜索”或检查仪表板/日志以确保内存消失。
</步骤>
</步骤>

## 删除单个内存（平台）

<代码组>```python Python
from mem0 import MemoryClient

client = MemoryClient(api_key="your-api-key")

memory_id = "your_memory_id"
client.delete(memory_id=memory_id)
```

```javascript JavaScript
import MemoryClient from 'mem0ai';

const client = new MemoryClient({ apiKey: "your-api-key" });

client.delete("your_memory_id")
  .then(result => console.log(result))
  .catch(error => console.error(error));
```
</代码组>

<信息图标=“检查”>
  您将收到确认有效负载。仪表板会在几秒钟内反映删除情况。
</信息>

## 批量删除多个内存（平台）

<代码组>```python Python
from mem0 import MemoryClient

client = MemoryClient(api_key="your-api-key")

delete_memories = [
    {"memory_id": "id1"},
    {"memory_id": "id2"}
]

response = client.batch_delete(delete_memories)
print(response)
```

```javascript JavaScript
import MemoryClient from 'mem0ai';

const client = new MemoryClient({ apiKey: "your-api-key" });

const deleteMemories = [
  { memory_id: "id1" },
  { memory_id: "id2" }
];

client.batchDelete(deleteMemories)
  .then(response => console.log('Batch delete response:', response))
  .catch(error => console.error(error));
```
</代码组>

## 通过过滤器删除内存（平台）

<代码组>```python Python
from mem0 import MemoryClient

client = MemoryClient(api_key="your-api-key")

# Delete all memories for a specific user
client.delete_all(user_id="alice")

# Delete all memories for a specific agent
client.delete_all(agent_id="support-bot")

# Delete all memories for a specific run
client.delete_all(run_id="session-xyz")
```

```javascript JavaScript
import MemoryClient from 'mem0ai';

const client = new MemoryClient({ apiKey: "your-api-key" });

client.deleteAll({ userId: "alice" })
  .then(result => console.log(result))
  .catch(error => console.error(error));
```
</代码组>

您还可以按其他参数进行过滤，例如：

- `agent_id`
- `run_id`
- `元数据`（作为 JSON 字符串）

<警告>
  **重大更改：** `delete_all` 之前在不使用过滤器的情况下调用时会擦除所有项目内存。如果没有提供过滤器，它现在**引发错误**。使用“*”通配符进行有意批量删除（见下文）。
</警告>

### 通配符删除

将过滤器设置为“*”会删除整个项目中该实体类型的**所有内存**。这是有意明确选择批量删除。

<代码组>```python Python
from mem0 import MemoryClient

client = MemoryClient(api_key="your-api-key")

# Delete all memories across every user in the project
client.delete_all(user_id="*")

# Delete all memories across every agent in the project
client.delete_all(agent_id="*")

# Full project wipe — all four filters must be explicitly set to "*"
client.delete_all(user_id="*", agent_id="*", app_id="*", run_id="*")
```

```javascript JavaScript
import MemoryClient from 'mem0ai';

const client = new MemoryClient({ apiKey: "your-api-key" });

// Delete all memories across every user in the project
client.deleteAll({ userId: "*" })
  .then(result => console.log(result))
  .catch(error => console.error(error));

// Full project wipe — all four filters must be explicitly set to "*"
client.deleteAll({ userId: "*", agentId: "*", appId: "*", runId: "*" })
  .then(result => console.log(result))
  .catch(error => console.error(error));
```
</代码组>

<警告>
  完整的项目擦除需要**所有四个**过滤器设置为“*”。仅将某些实体设置为“*”只会删除这些实体类型的内存，而不是整个项目。
</警告>

## 使用Mem0 OSS删除

<代码组>```python Python
from mem0 import Memory

memory = Memory()

memory.delete(memory_id="mem_123")
memory.delete_all(user_id="alice")
```
</代码组>

<注意>
  OSS JavaScript SDK 尚未公开删除帮助程序 - 在自托管时使用 REST API 或 Python SDK。
</注>

## 用例回顾

- 根据用户的要求忘记他们的偏好。
- 在过时或不正确的事实传播之前将其删除。
- 在会话到期或保留期限后清理记忆。
- 遵守隐私立法（GDPR、CCPA）和内部政策。

  **MCP 替代方案**：通过 <Link href="/platform/mem0-mcp">Mem0 MCP</Link>，AI 代理可以在数据变得不相关或应用户请求时删除自己的内存。
</标注>

## 方法比较

|方法|使用时 |需要身份证 |过滤器|
| ---| ---| ---| ---|
| `删除（内存_id）` |具体记录你知道吗| ✔️ | ✖️ |
| `batch_delete([...])` |您有一个要清除的 ID 列表 | ✔️ | ✖️ |
| `删除_全部(...)` |您需要忘记用户/代理/运行 | ✖️ | ✔️ |
