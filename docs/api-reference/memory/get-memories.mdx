列出由过滤器限定的内存以及分页结果。实体 ID（“user_id”、“agent_id”、“app_id”、“run_id”）**必须**在“filters”对象内传递 - 顶级实体 ID 会被拒绝，返回 400。

`filters` 对象支持复杂的逻辑运算（AND、OR、NOT）和比较运算符：

- `in`：匹配任何指定的值
- `gte`：大于或等于
- `lte`：小于或等于
- `gt`：大于
- `lt`：小于
- `ne`: 不等于
- `icontains`：不区分大小写的包含检查
- `*`：匹配所有内容的通配符

将“page”和“page_size”作为查询参数传递以对结果进行分页。

<代码组>```python Code
memories = client.get_all(
    filters={
        "AND": [
            {
                "user_id": "alex"
            },
            {
                "created_at": {"gte": "2024-07-01", "lte": "2024-07-31"}
            }
        ]
    },
    page=1,
    page_size=50
)```
