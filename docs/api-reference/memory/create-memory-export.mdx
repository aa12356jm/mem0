提交作业以使用可自定义的 Pydantic 模式创建结构化的内存导出。此过程可能需要一些时间才能完成，尤其是在导出大量内存时。您可以通过应用各种过滤器（例如，“user_id”、“agent_id”、“run_id”或“session_id”）并修改 Pydantic 架构来定制导出，以确保最终数据符合您的确切需求。
