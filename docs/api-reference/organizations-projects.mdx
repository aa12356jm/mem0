## 概述

组织和项目为 Mem0 平台提供多租户支持、访问控制和团队协作功能。使用这些 API 构建支持多个团队、客户或隔离环境的应用程序。

<信息>
组织和项目是**可选**功能。对于单用户或简单的多用户应用程序，您可以在没有它们的情况下使用 Mem0。
</信息>

## 关键能力

- **多组织/项目支持**：在初始化 Mem0 客户端时指定组织和项目，以适当地归因 API 使用情况
- **成员管理**：通过组织和项目成员身份控制对数据的访问
- **访问控制**：只有成员可以访问其组织/项目范围内的内存和数据
- **团队隔离**：保持不同团队和项目之间的数据分离，以实现安全协作

## 项目管理

Mem0 客户端通过“client.project”接口提供全面的项目管理：

### 获取项目详情

检索有关当前项目的信息：```python
# Get all project details
project_info = client.project.get()

# Get specific fields only
project_info = client.project.get(fields=["name", "description", "custom_categories"])```

### 创建一个新项目

在您的组织内创建一个新项目：

```python
# 创建一个项目并指定名称和描述
new_project = client.project.create(
    name="我的新项目",
    描述=“管理客户支持记忆的项目”
）```

### 更新项目设置

修改项目配置，包括自定义说明、类别、图形设置和语言首选项：

```python
# 使用自定义类别更新项目
客户端.项目.更新（
    自定义类别=[
        {"customer_preferences": "客户喜欢、不喜欢和偏好"},
        {"support_history": "以前的支持互动和解决方案"}
    ]
）

# 使用自定义指令更新项目
客户端.项目.更新(
    自定义指令=“...”
）

# 使用输入语言进行内存存储和检索
client.project.update(多语言=True)

# 一次更新多个设置
客户端.项目.更新(
    自定义指令=“...”，
    自定义类别=[
        {"personal_info": "用户个人信息和偏好"},
        {"work_context": "专业背景和工作相关信息"}
    ],
    多语言=正确
）```

### 删除项目

<警告>
此操作将删除项目中的所有记忆、消息和其他相关数据。 **此操作是不可逆的。**
</警告>

删除项目及其所有关联数据：

```python
# 删除当前项目（不可逆）
结果 = client.project.delete()```

## 会员管理

管理项目成员及其访问级别：

```python
# 获取所有项目成员
成员= client.project.get_members()

# 添加一个新成员作为读者
client.project.add_member(
    电子邮件=“colleague@company.com”，
    角色=“READER”#或“OWNER”
）

# 更新成员角色
client.project.update_member(
    电子邮件=“colleague@company.com”，
    角色=“所有者”
）

# 从项目中删除成员
client.project.remove_member（电子邮件=“colleague@company.com”）```

### 会员角色

| 角色 |权限|
|------|-------------|
| **读者** |可以查看和搜索记忆，但不能修改项目设置或管理成员 |
| **所有者** |完全访问权限，包括项目修改、成员管理和所有读者权限 |

## 异步支持

所有项目方法都可以在异步模式下使用：

```python
从 mem0 导入 AsyncMemoryClient

异步定义管理项目（）：
    客户端 = AsyncMemoryClient(api_key="your-api-key")

    # 所有方法都支持async/await
    项目信息 = 等待 client.project.get()
    等待 client.project.update(多语言=True)
    成员 = 等待 client.project.get_members()

# 正确调用异步函数
导入异步
asyncio.run（管理项目（））```
