mem0 中的 Config 是一个字典，用于指定嵌入模型的设置。它允许您自定义所选嵌入器的行为和连接详细信息。

## 如何定义配置？

配置被定义为具有两个主键的对象（或字典）：
- `embedder`：指定嵌入器提供程序及其配置
  - `provider`：嵌入者的名称（例如，“openai”、“ollama”）
  - `config`：包含特定于提供者的设置的嵌套对象或字典


## 如何使用配置？

以下是如何将配置与 mem0 一起使用的一般示例：

<代码组>```python Python
import os
from mem0 import Memory

os.environ["OPENAI_API_KEY"] = "sk-xx"

config = {
    "embedder": {
        "provider": "your_chosen_provider",
        "config": {
            # Provider-specific settings go here
        }
    }
}

m = Memory.from_config(config)
m.add("Your text here", user_id="user", metadata={"category": "example"})```

## 为什么需要配置？

配置对于以下方面至关重要：
1. 指定使用哪种嵌入模型。
2. 提供必要的连接详细信息（例如模型、api_key、embedding_dims）。
3. 确保正确初始化并连接到您选择的嵌入器。
