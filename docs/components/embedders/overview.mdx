## 用法要利用嵌入模型，您必须提供配置来自定义其用法。如果未提供配置，则将应用默认配置，并使用“OpenAI”作为嵌入模型。

有关嵌入模型配置的可用参数的完整列表，请参阅[Config](./config)。
