您可以使用 Ollama 的嵌入模型在本地运行 Mem0。

### 用法

<代码组>```python Python
import os
from mem0 import Memory

os.environ["OPENAI_API_KEY"] = "your_api_key" # For LLM

config = {
    "embedder": {
        "provider": "ollama",
        "config": {
            "model": "mxbai-embed-large"
        }
    }
}

m = Memory.from_config(config)
messages = [
    {"role": "user", "content": "I'm planning to watch a movie tonight. Any recommendations?"},
    {"role": "assistant", "content": "How about thriller movies? They can be quite engaging."},
    {"role": "user", "content": "I'm not a big fan of thriller movies but I love sci-fi movies."},
    {"role": "assistant", "content": "Got it! I'll avoid thriller recommendations and suggest sci-fi movies in the future."}
]
m.add(messages, user_id="john")
```

```typescript TypeScript
import { Memory } from 'mem0ai/oss';

const config = {
  embedder: {
    provider: 'ollama',
    config: {
      model: 'nomic-embed-text:latest', // or any other Ollama embedding model
      url: 'http://localhost:11434', // Ollama server URL
    },
  },
};

const memory = new Memory(config);
const messages = [
    {"role": "user", "content": "I'm planning to watch a movie tonight. Any recommendations?"},
    {"role": "assistant", "content": "How about thriller movies? They can be quite engaging."},
    {"role": "user", "content": "I'm not a big fan of thriller movies but I love sci-fi movies."},
    {"role": "assistant", "content": "Got it! I'll avoid thriller recommendations and suggest sci-fi movies in the future."}
]
await memory.add(messages, { userId: "john" });
```
</代码组>

### 配置

以下是可用于配置 Ollama 嵌入器的参数：

<标签>
<标签标题=“Python”>
|参数|描述 |默认值 |
| ---| ---| ---|
| `模型` |要使用的 Ollama 模型的名称 | `经济嵌入文本` |
| `embedding_dims` |嵌入模型的尺寸| `512` |
| `ollama_base_url` | ollama 连接的基本 URL | `无` |
</标签>
<选项卡标题=“TypeScript”>
|参数|描述 |默认值 |

| `网址` | Ollama 服务器的基本 URL | `http://localhost:11434` |
| `embeddingDims` |嵌入模型的尺寸|第768章
