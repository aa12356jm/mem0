您可以使用 Huggingface 中的嵌入模型在本地运行 Mem0。

＃＃＃ 用法```python
import os
from mem0 import Memory

os.environ["OPENAI_API_KEY"] = "your_api_key" # For LLM

config = {
    "embedder": {
        "provider": "huggingface",
        "config": {
            "model": "multi-qa-MiniLM-L6-cos-v1"
        }
    }
}

m = Memory.from_config(config)
messages = [
    {"role": "user", "content": "I'm planning to watch a movie tonight. Any recommendations?"},
    {"role": "assistant", "content": "How about thriller movies? They can be quite engaging."},
    {"role": "user", "content": "I'm not a big fan of thriller movies but I love sci-fi movies."},
    {"role": "assistant", "content": "Got it! I'll avoid thriller recommendations and suggest sci-fi movies in the future."}
]
m.add(messages, user_id="john")
```
### 使用文本嵌入推理 (TEI)

您还可以使用 Hugging Face 的文本嵌入推理服务来实现更快、更高效的嵌入：```python
import os
from mem0 import Memory

os.environ["OPENAI_API_KEY"] = "your_api_key" # For LLM

# Using HuggingFace Text Embeddings Inference API
config = {
    "embedder": {
        "provider": "huggingface",
        "config": {
            "huggingface_base_url": "http://localhost:3000/v1"
        }
    }
}

m = Memory.from_config(config)
m.add("This text will be embedded using the TEI service.", user_id="john")
```
要运行 TEI 服务，您可以使用 Docker：```bash
docker run -d -p 3000:80 -v huggingfacetei:/data --platform linux/amd64 \
    ghcr.io/huggingface/text-embeddings-inference:cpu-1.6 \
    --model-id BAAI/bge-small-en-v1.5
```
### 配置

以下是可用于配置 Huggingface 嵌入器的参数：

|参数|描述 |默认值 |
| ---| ---| ---|
| `模型` |要使用的模型名称 | `multi-qa-MiniLM-L6-cos-v1` |
| `embedding_dims` |嵌入模型的尺寸| `选定的模型尺寸` |
| `model_kwargs` |该模型的其他参数 | `无` |
| `huggingface_base_url` |连接到文本嵌入推理 (TEI) API 的 URL | `无` |
