Mem0支持LangChain作为提供者访问各种LLM模型。 LangChain是一个用于开发由语言模型支持的应用程序的框架，可以通过一致的接口轻松集成各种LLM提供商。

有关 LangChain 支持的可用聊天模型的完整列表，请参阅 [LangChain 聊天模型文档](https://python.langchain.com/docs/integrations/chat)。

## 用法

<代码组>```python Python
import os
from mem0 import Memory
from langchain_openai import ChatOpenAI

# Set necessary environment variables for your chosen LangChain provider
os.environ["OPENAI_API_KEY"] = "your-api-key"

# Initialize a LangChain model directly
openai_model = ChatOpenAI(
    model="gpt-5-mini",
    temperature=0.2,
    max_tokens=2000
)

# Pass the initialized model to the config
config = {
    "llm": {
        "provider": "langchain",
        "config": {
            "model": openai_model
        }
    }
}

m = Memory.from_config(config)
messages = [
    {"role": "user", "content": "I'm planning to watch a movie tonight. Any recommendations?"},
    {"role": "assistant", "content": "How about thriller movies? They can be quite engaging."},
    {"role": "user", "content": "I'm not a big fan of thriller movies but I love sci-fi movies."},
    {"role": "assistant", "content": "Got it! I'll avoid thriller recommendations and suggest sci-fi movies in the future."}
]
m.add(messages, user_id="alice", metadata={"category": "movies"})
```

```typescript TypeScript
import { Memory } from 'mem0ai/oss';
import { ChatOpenAI } from "@langchain/openai";

// Initialize a LangChain model directly
const openaiModel = new ChatOpenAI({
    modelName: "gpt-4",
    temperature: 0.2,
    maxTokens: 2000,
    apiKey: process.env.OPENAI_API_KEY,
});

const config = {
  llm: {
    provider: 'langchain',
    config: {
      model: openaiModel,
    },
  },
};

const memory = new Memory(config);
const messages = [
    {"role": "user", "content": "I'm planning to watch a movie tonight. Any recommendations?"},
    {"role": "assistant", "content": "How about thriller movies? They can be quite engaging."},
    {"role": "user", "content": "I'm not a big fan of thriller movies but I love sci-fi movies."},
    {"role": "assistant", "content": "Got it! I'll avoid thriller recommendations and suggest sci-fi movies in the future."}
]
await memory.add(messages, { userId: "alice", metadata: { category: "movies" } });
```

## 支持的浪链提供商

LangChain支持广泛的LLM提供商，包括：

- OpenAI（`ChatOpenAI`）
- 人择（`ChatAnthropic`）
- 谷歌（`ChatGoogleGenerativeAI`、`ChatGooglePalm`）
- 米斯特拉尔（`ChatMistralAI`）
- Ollama（`ChatOllama`）
- Azure OpenAI（`AzureChatOpenAI`）
- HuggingFace (`HuggingFaceChatEndpoint`)
- 还有更多

您可以直接在配置中使用这些模型实例中的任何一个。有关可用提供商的完整最新列表，请参阅 [LangChain 聊天模型文档](https://python.langchain.com/docs/integrations/chat)。

## 特定于提供商的配置

当使用LangChain作为提供商时，您需要：

1. 为您选择的LLM提供商设置适当的环境变量
2.导入并初始化您要使用的具体模型类
3.将初始化的模型实例传递给配置

<注意>
  确保安装必要的 LangChain 软件包和任何特定于提供商的依赖项。
</注>
