## 用法

要使用 llm，您必须提供配置来自定义其用法。如果未提供配置，将应用默认配置，并使用“OpenAI”作为 llm。

有关 llm 配置的可用参数的完整列表，请参阅 [Config](./config)。

## 结构化与非结构化输出Mem0 支持两种类型的 OpenAI LLM 格式，每种格式都有自己的优势和用例：

### 结构化输出

结构化输出是与 OpenAI 结构化输出模型一致的法学硕士：

- **优化：** 返回结构化响应（例如 JSON 对象）
- **优点：** 精确、易于解析的数据
- **非常适合：** 数据提取、表单填写、API 响应
- **了解更多：** [OpenAI 结构化输出指南](https://platform.openai.com/docs/guides/structed-outputs/introduction）

### 非结构化输出

非结构化输出对应于 OpenAI 的标准、自由格式文本模型：

- **灵活性：** 返回开放式的自然语言响应
- **定制：** 使用 `response_format` 参数来指导输出
- **权衡：** 与满足特定数据需求的结构化输出相比，效率较低
- **最适合：** 创意写作、解释、一般对话

选择最适合您的应用程序要求的格式，以获得最佳性能和可用性。
