Sentence Transformer reranker 使用 HuggingFace 交叉编码器模型提供本地重新排名，非常适合您希望将数据保留在本地的注重隐私的部署。

## 型号

可以使用任何 HuggingFace 交叉编码器模型。受欢迎的选择包括：

- **`cross-encoder/ms-marco-MiniLM-L-6-v2`**：默认，速度和精度的良好平衡
- **`cross-encoder/ms-marco-TinyBERT-L-2-v2`**：最快、更小的模型尺寸
- **`cross-encoder/ms-marco-electra-base`**：精度更高，模型更大
- **`cross-encoder/stsb-distilroberta-base`**：适合语义相似性任务

```
## 使用示例```python Python
from mem0 import Memory

# Initialize memory with local reranker
config = {
    "vector_store": {"provider": "chroma"},
    "llm": {"provider": "openai", "config": {"model": "gpt-4o-mini"}},
    "rerank": {
        "provider": "sentence_transformer",
        "config": {
            "model": "cross-encoder/ms-marco-MiniLM-L-6-v2",
            "device": "cpu"
        }
    }
}

memory = Memory.from_config(config)

# Add memories
messages = [
    {"role": "user", "content": "I love reading science fiction novels"},
    {"role": "user", "content": "My favorite author is Isaac Asimov"},
    {"role": "user", "content": "I also enjoy watching sci-fi movies"}
]

memory.add(messages, user_id="charlie")

# Search with local reranking
results = memory.search("What books does the user like?", filters={"user_id": "charlie"})

for result in results['results']:
    print(f"Memory: {result['memory']}")
    print(f"Vector Score: {result['score']:.3f}")
    print(f"Rerank Score: {result['rerank_score']:.3f}")
    print()
```
## 定制模型

您可以使用任何 HuggingFace 交叉编码器模型：```python Python
# Using a different model
config = {
    "rerank": {
        "provider": "sentence_transformer", 
        "config": {
            "model": "cross-encoder/stsb-distilroberta-base",
            "device": "cpu"
        }
    }
}
```
## 配置参数

|参数|描述 |类型 |默认|
|------------|-------------|------|---------|
| `模型` | HuggingFace 交叉编码器模型名称 | `str` | `“交叉编码器/ms-marco-MiniLM-L-6-v2”` |
| `设备` |运行模型的设备（“cpu”、“cuda”等）)| `str` | `无` |
| `batch_size` |处理文件的批量大小 | `int` | `32` |
| `显示进度条` |处理过程中显示进度条 | `布尔` | ‘假’|
| `top_k` |返回的最大文件数 | `int` | `无` |

## 优点

- **隐私**：完全本地处理，无外部API调用
- **成本**：初始模型下载后，不再按代币收费
- **定制**：使用任何 HuggingFace 交叉编码器模型
- **离线**：模型下载后无需互联网连接即可工作

## 性能考虑因素

- **首次运行**：模型下载最初可能需要一些时间
- **内存使用**：模型需要 GPU/CPU 内存
- **批量大小**：根据可用内存优化批量大小
- **设备**：GPU加速显着提高速度

## 最佳实践1. **模型选择**：根据精度与速度要求选择模型
2. **设备管理**：在可用时使用 GPU 以获得更好的性能
3. **批处理**：同时处理多个文档以提高效率
4. **内存监控**：监控较大模型的系统内存使用情况
