## 概述

Hugging Face 重新排序器提供商使您可以访问 Hugging Face Hub 上提供的数千个重新排序模型。这包括流行的模型，如 BAAI 的 BGE 重新排序器和其他最先进的交叉编码器模型。

## 配置

### 基本设置```python
from mem0 import Memory

config = {
    "reranker": {
        "provider": "huggingface",
        "config": {
            "model": "BAAI/bge-reranker-base",
            "device": "cpu"
        }
    }
}

m = Memory.from_config(config)
```
### 配置参数

|参数|类型 |默认|描述 |
|------------|------|---------|------------|
| `模型` | STR |必填 |拥抱脸模型标识符|
| `设备` | STR | “CPU” |运行模型的设备（“cpu”、“cuda”、“mps”）|
| `batch_size` |整数 | 32 | 32加工批量大小|
| `最大长度` |整数 | 512 | 512最大输入序列长度|
| `信任远程代码` |布尔 |假|允许远程执行代码 |

### 高级配置```python

## 用法示例

### 基本用法```python
from mem0 import Memory

m = Memory.from_config(config)

# Add some memories
m.add("I love hiking in the mountains", user_id="alice")
m.add("Pizza is my favorite food", user_id="alice")
m.add("I enjoy reading science fiction books", user_id="alice")

# Search with reranking
results = m.search(
    "What outdoor activities do I enjoy?",
    user_id="alice",
    rerank=True
)

for result in results["results"]:
    print(f"Memory: {result['memory']}")
    print(f"Score: {result['score']:.3f}")
```
### 批处理```python
# Process multiple queries efficiently
queries = [
    "What are my hobbies?",
    "What food do I like?",
    "What books interest me?"
]

results = []
for query in queries:
    result = m.search(query, filters={"user_id": "alice"}, rerank=True)
    results.append(result)
```

## 错误处理```python
try:
    results = m.search(
        "test query",
        user_id="alice",
        rerank=True
    )
except Exception as e:
    print(f"Reranking failed: {e}")
    # Fall back to vector search only
    results = m.search(
        "test query",
        user_id="alice",
        rerank=False
    )
```

## 最佳实践

1. **选择正确的型号**：根据您的需求平衡质量与速度
2. **使用 GPU**：对于较大的模型，速度明显快于 CPU
3. **优化批量大小**：根据您的硬件能力进行调整
4. **监控内存**：观察大型模型的 GPU/CPU 内存使用情况
5. **缓存模型**：下载一次，重复使用，避免重复下载

## 故障排除

### 常见问题

**内存不足错误**```python
# Reduce batch size and sequence length
config = {
    "reranker": {
        "provider": "huggingface",
        "config": {
            "model": "BAAI/bge-reranker-base",
            "batch_size": 4,
            "max_length": 256
        }
    }
}
```
**模型下载问题**```python
# Set cache directory
import os
os.environ["TRANSFORMERS_CACHE"] = "/path/to/cache"

# Or use offline mode
config = {
    "reranker": {
        "provider": "huggingface",
        "config": {
            "model": "BAAI/bge-reranker-base",
            "local_files_only": True
        }
    }
}
```
**CUDA 不可用**```python
import torch

config = {
    "reranker": {
        "provider": "huggingface",
        "config": {
            "model": "BAAI/bge-reranker-base",
            "device": "cuda" if torch.cuda.is_available() else "cpu"
        }
    }
}
```
