Cohere 提供企业级重新排名模型，具有出色的多语言支持和生产就绪性能。

## 型号

Cohere 提供了几种重新排名模型：

- **`rerank-english-v3.0`**：具有最佳性能的最新英语重新排序器
- **`rerank-multilingual-v3.0`**：全球应用程序的多语言支持
- **`rerank-english-v2.0`**：上一代英语重新排序器

## 安装```bash
pip install cohere
```
＃＃ 配置```python Python
from mem0 import Memory

config = {
    "vector_store": {
        "provider": "chroma",
        "config": {
            "collection_name": "my_memories",
            "path": "./chroma_db"
        }
    },
    "llm": {
        "provider": "openai",
        "config": {
            "model": "gpt-5-mini"
        }
    },
    "reranker": {
        "provider": "cohere",
        "config": {
            "model": "rerank-english-v3.0",
            "api_key": "your-cohere-api-key",  # or set COHERE_API_KEY
            "top_k": 5,
            "return_documents": False,
            "max_chunks_per_doc": None
        }
    }
}

memory = Memory.from_config(config)
```

```
## 使用示例```python Python
import os
from mem0 import Memory

# Set API key
os.environ["COHERE_API_KEY"] = "your-api-key"

# Initialize memory with Cohere reranker
config = {
    "vector_store": {"provider": "chroma"},
    "llm": {"provider": "openai", "config": {"model": "gpt-4o-mini"}},
    "rerank": {
        "provider": "cohere",
        "config": {
            "model": "rerank-english-v3.0",
            "top_k": 3
        }
    }
}

memory = Memory.from_config(config)

# Add memories
messages = [
    {"role": "user", "content": "I work as a data scientist at Microsoft"},
    {"role": "user", "content": "I specialize in machine learning and NLP"},
    {"role": "user", "content": "I enjoy playing tennis on weekends"}
]

memory.add(messages, user_id="bob")

# Search with reranking
results = memory.search("What is the user's profession?", filters={"user_id": "bob"})

for result in results['results']:
    print(f"Memory: {result['memory']}")
    print(f"Vector Score: {result['score']:.3f}")
    print(f"Rerank Score: {result['rerank_score']:.3f}")
    print()
```
## 多语言支持

对于多语言应用程序，请使用多语言模型：```python Python
config = {
    "rerank": {
        "provider": "cohere",
        "config": {
            "model": "rerank-multilingual-v3.0",
            "top_k": 5
        }
    }
}
```
## 配置参数

|参数|描述 |类型 |默认 |
| -------------------- | -------------------------------- | ------ | ----------------------- |
| `模型` |使用 Cohere 重新排序模型 | `str` | `"rerank-english-v3.0"` |
| `api_key` | Cohere API 密钥 | `str` | `无` |
| `top_k` |返回的最大文件数 | `int` | `无` |
| `return_documents` |是否返回文档文本 | `布尔` | ‘假’|
| `max_chunks_per_doc` |每个文档的最大块数 | `int` | `无` |
