## 常用配置参数

所有重新排序器共享这些通用配置参数：

|参数|描述 |类型 |默认|
| ---------- | --------------------------------------------------- | -----| -------- |
| `提供者` |重新排序提供商名称 | `str` |必填 |
| `top_k` |重新排名后返回的最大结果数 | `int` | `无` |
| `api_key` |重新排序服务的 API 密钥 | `str` | `无` |

## 特定于提供商的配置

### 零熵

|参数|描述 |类型 |默认 |
| ---------| -------------------------------------------------------- | -----| ------------ |
| `模型` |使用的模型：`zerank-1` 或 `zerank-1-small` | `str` | `“zerank-1”` |
| `api_key` |零熵 API 密钥 | `str` | `无` |

### 连贯|参数|描述 |类型 |默认 |
| -------------------- | -------------------------------------------------------- | ------ | ----------------------- |
| `模型` | Cohere 重新排序模型 | `str` | `"rerank-english-v3.0"` |
| `api_key` | Cohere API 密钥 | `str` | `无` |
| `return_documents` |是否返回文档文本作为响应 | `布尔` | ‘假’|
| `max_chunks_per_doc` |每个文档的最大块数 | `int` | `无` |

### 句子转换器|参数|描述 |类型 |默认 |
| ------------------- | -------------------------------------------------------- | ------ | ---------------------------------------------------- |
| `模型` | HuggingFace 交叉编码器模型名称 | `str` | `“交叉编码器/ms-marco-MiniLM-L-6-v2”` |
| `设备` |运行模型的设备（`cpu`、`cuda` 等）| `str` | `无` |
| `batch_size` |加工批量大小 | `int` | `32` |
| `显示进度条` |显示处理过程中的进度 | `布尔` | ‘假’|

### 抱脸|参数|描述 |类型 |默认 |
| ---------| -------------------------------------------------------- | -----| ------------------------ | |
| `模型` | HuggingFace 重新排序模型名称 | `str` | `"BAAI/bge-reranker-large"` |
| `api_key` | HuggingFace API 令牌 | `str` | `无` |
| `设备` |运行模型的设备（`cpu`、`cuda` 等）| `str` | `无` |

### 基于法学硕士|参数|描述 |类型 |默认 |
| ---------------- | ------------------------------------------------------ | -------- | ---------------------- |
| `模型` |用于评分的LLM模型| `str` | `"gpt-4o-mini"` |
| `提供者` | LLM 提供商（`openai`、`anthropic` 等）| `str` | `"openai"` |
| `api_key` | LLM 提供商的 API 密钥 | `str` | `无` |
| `温度` | LLM 生成温度 | `浮动` | `0。0` |
| `max_tokens` | LLM 响应的最大令牌数 | `int` | `100` |
| `scoring_prompt` |自定义评分提示模板| `str` |默认评分提示 |

### LLM 重新排名

|参数|描述 |类型 |默认|
| -------------- | ------------------------ | | ------ | -------- |
| `llm.provider` |重新排名的LLM提供商| `str` |必填 |
| `llm.config` | LLM配置对象| `字典` |必填 |
| `top_n` |返回的结果数 | `int` | `无` |

## 环境变量您可以使用环境变量设置 API 密钥：

- `ZERO_ENTROPY_API_KEY` - 零熵 API 密钥
- `COHERE_API_KEY` - Cohere API 密钥
- `HUGGINGFACE_API_KEY` - HuggingFace API 令牌
- `OPENAI_API_KEY` - OpenAI API 密钥（用于基于 LLM 的重新排名器）
- `ANTHROPIC_API_KEY` - Anthropic API 密钥（用于基于 LLM 的重新排名器）

## 基本配置示例```python Python
config = {
    "vector_store": {
        "provider": "chroma",
        "config": {
            "collection_name": "my_memories",
            "path": "./chroma_db"
        }
    },
    "llm": {
        "provider": "openai",
        "config": {
            "model": "gpt-5-mini"
        }
    },
    "reranker": {
        "provider": "zero_entropy",
        "config": {
            "model": "zerank-1",
            "top_k": 5
        }
    }
}
```
