要利用矢量数据库，您必须提供配置来自定义其使用。如果未提供配置，则将应用默认配置，并且“Qdrant”将用作矢量数据库。

有关矢量数据库配置的可用参数的完整列表，请参阅[Config](./config)。

## 常见问题

### 使用不同尺寸的模型

如果您使用的是1536以外的其他尺寸的定制模型（例如768），您可能会遇到以下错误：

`ValueError：形状 (0,1536) 和 (768,) 未对齐：1536 (dim 1) != 768 (dim 0)`

您可以将 `"embedding_model_dims": 768,` 添加到矢量存储的配置中来解决此问题。
