Mem0 支持 LangChain 作为矢量存储集成的提供者。 LangChain为各种矢量数据库提供了统一的接口，可以通过一致的API轻松集成不同的矢量存储提供商。

<注意>
  当使用 LangChain 作为矢量存储提供商时，您必须将集合名称设置为“mem0”。这是与 Mem0 正确集成所需的配置。
</注>

## 用法

<代码组>```python Python
import os
from mem0 import Memory
from langchain_chroma import Chroma
from langchain_openai import OpenAIEmbeddings

# Initialize a LangChain vector store
embeddings = OpenAIEmbeddings()
vector_store = Chroma(
    persist_directory="./chroma_db",
    embedding_function=embeddings,
    collection_name="mem0"  # Required collection name
)

# Pass the initialized vector store to the config
config = {
    "vector_store": {
        "provider": "langchain",
        "config": {
            "client": vector_store
        }
    }
}

m = Memory.from_config(config)
messages = [
    {"role": "user", "content": "I'm planning to watch a movie tonight. Any recommendations?"},
    {"role": "assistant", "content": "How about thriller movies? They can be quite engaging."},
    {"role": "user", "content": "I'm not a big fan of thriller movies but I love sci-fi movies."},
    {"role": "assistant", "content": "Got it! I'll avoid thriller recommendations and suggest sci-fi movies in the future."}
]
m.add(messages, user_id="alice", metadata={"category": "movies"})
```

## 支持的 LangChain 矢量存储

LangChain支持广泛的矢量商店提供商，包括：

- 色度
- 费斯
- 松果
- 维维特
- 米尔乌斯
-Qdrant
- 还有更多

您可以直接在配置中使用这些矢量存储实例中的任何一个。有关可用提供程序的完整最新列表，请参阅 [LangChain Vector Stores 文档](https://python.langchain.com/docs/integrations/vectorstores)。

## 限制

当使用 LangChain 作为矢量存储提供商时，需要注意一些限制：

1. **批量操作**：使用 LangChain 作为向量存储提供者时，不支持 `get_all` 和 `delete_all` 操作。这是因为 LangChain 的向量存储接口没有为所有提供商的这些批量操作提供标准化方法。

2.**特定于提供商的功能**：根据您通过 LangChain 使用的特定矢量存储实现，某些高级功能可能不可用。

## 特定于提供商的配置

当使用 LangChain 作为矢量存储提供商时，您需要：1. 为您选择的矢量存储提供商设置适当的环境变量
2. 导入并初始化您要使用的特定向量存储类
3. 将初始化的向量存储实例传递给配置

<注意>
  确保安装必要的 LangChain 软件包和任何特定于提供商的依赖项。
</注>
