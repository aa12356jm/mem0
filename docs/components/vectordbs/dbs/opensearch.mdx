[OpenSearch](https://opensearch.org/) 是一个企业级搜索和可观察性套件，可大规模整理非结构化数据。 OpenSearch 支持 k-NN（k-最近邻）并允许您有效地存储和检索高维向量嵌入。

### 安装

OpenSearch 支持需要额外的依赖项。安装它们：```bash
pip install opensearch-py
```
### 先决条件

在将 OpenSearch 与 Mem0 结合使用之前，您需要在 AWS OpenSearch Service 中设置集合。

#### AWS OpenSearch 服务
您可以通过 AWS 控制台创建集合：
- 导航到 [OpenSearch 服务控制台](https://console.aws.amazon.com/aos/home)
- 点击“创建收藏”
- 选择“无服务器集合”，然后启用“矢量搜索”功能
- 创建后，记下您的配置的端点 URL（主机）


### 用法```python
import os
from mem0 import Memory
import boto3
from opensearchpy import OpenSearch, RequestsHttpConnection, AWSV4SignerAuth

# For AWS OpenSearch Service with IAM authentication
region = 'us-west-2'
service = 'aoss'
credentials = boto3.Session().get_credentials()
auth = AWSV4SignerAuth(credentials, region, service)

config = {
    "vector_store": {
        "provider": "opensearch",
        "config": {
            "collection_name": "mem0",
            "host": "your-domain.us-west-2.aoss.amazonaws.com",
            "port": 443,
            "http_auth": auth,
            "embedding_model_dims": 1024,
            "connection_class": RequestsHttpConnection,
            "pool_maxsize": 20,
            "use_ssl": True,
            "verify_certs": True
        }
    }
}
```
### 添加回忆```python
m = Memory.from_config(config)
messages = [
    {"role": "user", "content": "I'm planning to watch a movie tonight. Any recommendations?"},
    {"role": "assistant", "content": "How about thriller movies? They can be quite engaging."},
    {"role": "user", "content": "I'm not a big fan of thriller movies but I love sci-fi movies."},
    {"role": "assistant", "content": "Got it! I'll avoid thriller recommendations and suggest sci-fi movies in the future."}
]
m.add(messages, user_id="alice", metadata={"category": "movies"})
```
### 搜索记忆```python
results = m.search("What kind of movies does Alice like?", filters={"user_id": "alice"})
```
### 特点

- 快速高效的矢量搜索
- 可以部署在本地、容器中或 AWS OpenSearch Service 等云平台上
- 多种身份验证和安全方法（基本身份验证、API 密钥、LDAP、SAML 和 OpenID Connect）
- 通过矢量搜索的优化映射自动创建索引
- 通过基于磁盘的矢量搜索和量化进行内存优化
- 实时分析和可观察性
