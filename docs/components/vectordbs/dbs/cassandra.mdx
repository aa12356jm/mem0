[Apache Cassandra](https://cassandra.apache.org/) 是一种高度可扩展的分布式 NoSQL 数据库，旨在跨多个商用服务器处理大量数据，且不会出现单点故障。它支持人工智能应用中语义搜索功能的矢量存储，并且可以通过线性性能改进扩展到海量数据集。

＃＃＃ 用法```python
import os
from mem0 import Memory

os.environ["OPENAI_API_KEY"] = "sk-xx"

config = {
    "vector_store": {
        "provider": "cassandra",
        "config": {
            "contact_points": ["127.0.0.1"],
            "port": 9042,
            "username": "cassandra",
            "password": "cassandra",
            "keyspace": "mem0",
            "collection_name": "memories",
        }
    }
}

m = Memory.from_config(config)
messages = [
    {"role": "user", "content": "I'm planning to watch a movie tonight. Any recommendations?"},
    {"role": "assistant", "content": "How about thriller movies? They can be quite engaging."},
    {"role": "user", "content": "I'm not a big fan of thriller movies but I love sci-fi movies."},
    {"role": "assistant", "content": "Got it! I'll avoid thriller recommendations and suggest sci-fi movies in the future."}
]
m.add(messages, user_id="alice", metadata={"category": "movies"})
```

### 配置

以下是可用于配置 Apache Cassandra 的参数：

|参数|描述 |默认值 |
| ---| ---| ---|
| `接触点` |联系点 IP 地址列表 |必填 |
| `端口` |卡桑德拉端口| `9042` |
| `用户名` |数据库用户名| `无` |
| `密码` |数据库密码| `无` |
| `键空间` |键空间名称 | `"mem0"` |
| `集合名称` |存储向量的表名 | `“回忆”` |
| `embedding_model_dims` |嵌入向量的维度 | `1536` |
| `secure_connect_bundle` | Astra DB 安全连接包的路径 | `无` |
| `协议版本` | CQL协议版本 | `4` |
| `load_balancing_policy` |自定义负载均衡策略| `无` |

### 设置

#### 选项 1：使用 Docker 进行本地 Cassandra 设置：```bash
# Pull and run Cassandra container
docker run --name mem0-cassandra \
    -p 9042:9042 \
    -e CASSANDRA_CLUSTER_NAME="Mem0Cluster" \
    -d cassandra:latest

# Wait for Cassandra to start (may take 1-2 minutes)
docker exec -it mem0-cassandra cqlsh

# Create keyspace
CREATE KEYSPACE IF NOT EXISTS mem0
WITH replication = {'class': 'SimpleStrategy', 'replication_factor': 1};
```

#### 选项 3：本地安装 Cassandra：

**Ubuntu/Debian：**```bash
# Add Apache Cassandra repository
echo "deb https://downloads.apache.org/cassandra/debian 40x main" | sudo tee -a /etc/apt/sources.list.d/cassandra.sources.list
curl https://downloads.apache.org/cassandra/KEYS | sudo apt-key add -

# Install Cassandra
sudo apt-get update
sudo apt-get install cassandra

# Start Cassandra
sudo systemctl start cassandra

# Verify installation
nodetool status
```
**苹果系统：**```bash
# Using Homebrew
brew install cassandra

# Start Cassandra
brew services start cassandra

# Connect to CQL shell
cqlsh
```

### 性能考虑因素

- **复制因子**：对于生产，使用至少 3 的复制因子
- **一致性级别**：一致性和性能之间的平衡（推荐QUORUM）
- **分区**：Cassandra 自动跨节点分配数据
- **扩展**：添加节点以线性增加容量和性能

### 高级配置```python
from cassandra.policies import DCAwareRoundRobinPolicy

config = {
    "vector_store": {
        "provider": "cassandra",
        "config": {
            "contact_points": ["node1.example.com", "node2.example.com", "node3.example.com"],
            "port": 9042,
            "username": "mem0_user",
            "password": "secure_password",
            "keyspace": "mem0_prod",
            "collection_name": "memories",
            "protocol_version": 4,
            "load_balancing_policy": DCAwareRoundRobinPolicy(local_dc='DC1')
        }
    }
}
```
<警告>
对于生产使用，请根据您的可用性和一致性要求配置适当的复制策略和一致性级别。
</警告>
