[Pinecone](https://www.pinecone.io/) 是一个完全托管的矢量数据库，专为机器学习应用程序而设计，提供大规模低延迟的高性能矢量搜索。它特别适合语义搜索、推荐系统和其他人工智能驱动的应用程序。

> **新**：Pinecone 集成现在支持自定义命名空间！使用“namespace”参数在逻辑上分隔同一索引中的数据。这对于多租户或多用户应用程序特别有用。

> **注意**：在配置 Pinecone 之前，您需要选择一个嵌入模型（例如 OpenAI、Cohere 或自定义模型），并确保配置中的“embedding_model_dims”与您选择的模型的尺寸相匹配。例如，OpenAI 的 text-embedding-3-small 使用 1536 维。

＃＃＃ 用法```python
import os
from mem0 import Memory

os.environ["OPENAI_API_KEY"] = "sk-xx"
os.environ["PINECONE_API_KEY"] = "your-api-key"

# Example using serverless configuration
config = {
    "vector_store": {
        "provider": "pinecone",
        "config": {
            "collection_name": "testing",
            "embedding_model_dims": 1536,  # Matches OpenAI's text-embedding-3-small
            "namespace": "my-namespace", # Optional: specify a namespace for multi-tenancy
            "serverless_config": {
                "cloud": "aws",  # Choose between 'aws' or 'gcp' or 'azure'
                "region": "us-east-1"
            },
            "metric": "cosine"
        }
    }
}

m = Memory.from_config(config)
messages = [
    {"role": "user", "content": "I'm planning to watch a movie tonight. Any recommendations?"},
    {"role": "assistant", "content": "How about thriller movies? They can be quite engaging."},
    {"role": "user", "content": "I'm not a big fan of thriller movies but I love sci-fi movies."},
    {"role": "assistant", "content": "Got it! I'll avoid thriller recommendations and suggest sci-fi movies in the future."}
]
m.add(messages, user_id="alice", metadata={"category": "movies"})
```
### 配置

以下是可用于配置 Pinecone 的参数：

|参数|描述 |默认值 |
| ---| ---| ---|
| `集合名称` |索引/集合的名称 |必填 |
| `embedding_model_dims` |嵌入模型的尺寸（必须与您选择的嵌入模型相匹配）|必填 |
| `客户` |现有的 Pinecone 客户端实例 | `无` |
| `api_key` | Pinecone 的 API 密钥 |环境变量：`PINECONE_API_KEY` |
| `环境` |松果环境| `无` |
| `无服务器配置` |无服务器部署的配置（AWS 或 GCP 或 Azure）| `无` |
| `pod_config` |基于 Pod 的部署配置 | `无` |
| `混合搜索` |是否启用混合搜索 | ‘假’|
| `公制` |向量相似度的距离度量 | `“余弦”` |
| `batch_size` |操作批量大小 | `100` |
| `命名空间` |集合的命名空间，对于多租户很有用。 | `无` |

> **重要**：您必须为部署选择 `serverless_config` 或 `pod_config`，但不能同时选择两者。

#### 无服务器配置示例```python
config = {
    "vector_store": {
        "provider": "pinecone",
        "config": {
            "collection_name": "memory_index",
            "embedding_model_dims": 1536,  # For OpenAI's text-embedding-3-small
            "namespace": "my-namespace",  # Optional: custom namespace
            "serverless_config": {
                "cloud": "aws",  # or "gcp" or "azure"
                "region": "us-east-1"  # Choose appropriate region
            }
        }
    }
}
```
#### Pod 配置示例```python
config = {
    "vector_store": {
        "provider": "pinecone",
        "config": {
            "collection_name": "memory_index",
            "embedding_model_dims": 1536,  # For OpenAI's text-embedding-ada-002
            "namespace": "my-namespace",  # Optional: custom namespace
            "pod_config": {
                "environment": "gcp-starter",
                "replicas": 1,
                "pod_type": "starter"
            }
        }
    }
}
```
