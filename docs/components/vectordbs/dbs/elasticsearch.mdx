[Elasticsearch](https://www.elastic.co/) 是一种分布式 RESTful 搜索和分析引擎，可以使用密集向量和 k-NN 搜索高效存储和搜索向量数据。

### 安装

Elasticsearch 支持需要额外的依赖项。安装它们：```bash
pip install elasticsearch>=8.0.0
```
＃＃＃ 用法```python
import os
from mem0 import Memory

os.environ["OPENAI_API_KEY"] = "sk-xx"

config = {
    "vector_store": {
        "provider": "elasticsearch",
        "config": {
            "collection_name": "mem0",
            "host": "localhost",
            "port": 9200,
            "embedding_model_dims": 1536
        }
    }
}

m = Memory.from_config(config)
messages = [
    {"role": "user", "content": "I'm planning to watch a movie tonight. Any recommendations?"},
    {"role": "assistant", "content": "How about thriller movies? They can be quite engaging."},
    {"role": "user", "content": "I’m not a big fan of thriller movies but I love sci-fi movies."},
    {"role": "assistant", "content": "Got it! I'll avoid thriller recommendations and suggest sci-fi movies in the future."}
]
m.add(messages, user_id="alice", metadata={"category": "movies"})
```
### 配置

以下是可用于配置 Elasticsearch 的参数：|参数|描述 |默认值 |
| ---------------------- | -------------------------------------------------- | ------------- |
| `集合名称` |存储向量的索引名称 | `mem0` |
| `embedding_model_dims` |嵌入模型的尺寸 | `1536` |
| `主机` |运行Elasticsearch服务器的主机 | `本地主机` |
| `端口` | Elasticsearch 服务器运行的端口 | `9200` |
| `云ID` |用于弹性云部署的云 ID | `无` |
| `api_key` |用于身份验证的 API 密钥 | `无` |
| `用户` |基本身份验证的用户名 | `无` |
| `密码` |基本身份验证的密码 | `无` |
| `验证证书` |是否验证SSL证书 | '真实' |
| `自动创建索引` |是否自动创建索引 | '真实' |
| `自定义搜索查询` |返回自定义搜索查询的函数 | `无` |
| `标题` |要包含在请求中的自定义标头 | `无` |### 特点

- 使用 Elasticsearch 的原生 k-NN 搜索进行高效矢量搜索
- 支持本地和云部署（弹性云）
- 多种身份验证方法（基本身份验证、API 密钥）
- 通过矢量搜索的优化映射自动创建索引
- 通过有效负载过滤进行内存隔离
- 自定义搜索查询功能，自定义搜索查询

### 自定义搜索查询

`custom_search_query` 参数允许您在调用 `Memory.search` 时自定义搜索查询。  
  
__例子__```python
import os
from typing import List, Optional, Dict
from mem0 import Memory

def custom_search_query(query: List[float], limit: int, filters: Optional[Dict]) -> Dict:
    return {
        "knn": {
            "field": "vector", 
            "query_vector": query, 
            "k": limit, 
            "num_candidates": limit * 2
        }
    }

os.environ["OPENAI_API_KEY"] = "sk-xx"

config = {
    "vector_store": {
        "provider": "elasticsearch",
        "config": {
            "collection_name": "mem0",
            "host": "localhost",
            "port": 9200,
            "embedding_model_dims": 1536,
            "custom_search_query": custom_search_query
        }
    }
}
```
它应该是一个带有以下参数的函数：
- `query`：`Memory.search` 中使用的查询向量
- `limit`：`Memory.search` 中使用的结果数量
- `filters`：`Memory.search` 中使用的键值对字典。您可以为自定义搜索查询添加自定义对。  
  
该函数应返回 Elasticsearch 搜索 API 的查询正文。
