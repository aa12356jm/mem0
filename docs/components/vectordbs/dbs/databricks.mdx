[Databricks 矢量搜索](https://docs.databricks.com/en/generative-ai/vector-search.html) 是一个无服务器相似性搜索引擎，可让您在矢量数据库中存储数据的矢量表示（包括元数据）。通过矢量搜索，您可以从 Unity Catalog 管理的 Delta 表创建自动更新的矢量搜索索引，并使用简单的 API 查询它们以返回最相似的矢量。

＃＃＃ 用法```python
import os
from mem0 import Memory

config = {
    "vector_store": {
        "provider": "databricks",
        "config": {
            "workspace_url": "https://your-workspace.databricks.com",
            "access_token": "your-access-token",
            "endpoint_name": "your-vector-search-endpoint",
            "catalog": "your_catalog",
            "schema": "your_schema",
            "table_name": "your_table",
            "collection_name": "your_index_name",
            "embedding_dimension": 1536
        }
    }
}

m = Memory.from_config(config)
messages = [
    {"role": "user", "content": "I'm planning to watch a movie tonight. Any recommendations?"},
    {"role": "assistant", "content": "How about thriller movies? They can be quite engaging."},
    {"role": "user", "content": "I'm not a big fan of thriller movies but I love sci-fi movies."},
    {"role": "assistant", "content": "Got it! I'll avoid thriller recommendations and suggest sci-fi movies in the future."}
]
m.add(messages, user_id="alice", metadata={"category": "movies"})
```
### 配置

以下是可用于配置 Databricks 矢量搜索的参数：|参数|描述 |默认值 |
| ---| ---| ---|
| `workspace_url` | Databricks 工作区的 URL | **必填** |
| `access_token` |用于身份验证的个人访问令牌 | `无` |
| `client_id` |服务主体客户端 ID（access_token 的替代方案）| `无` |
| `client_secret` |服务主体客户端密钥（需要与 client_id 一起使用）| `无` |
| `azure_client_id` | Azure AD 应用程序客户端 ID（用于 Azure Databricks）| `无` |
| `azure_client_secret` | Azure AD 应用程序客户端密钥（用于 Azure Databricks） | `无` |
| `端点名称` |矢量搜索端点的名称 | **必填** |
| `目录` | Unity Catalog 目录名称 | **必填** |
| `模式` | Unity 目录架构名称 | **必填** |
| `表名` |源 Delta 表名称 | **必填** |
| `集合名称` |矢量搜索索引名称 | `mem0` |
| `索引类型` |索引类型：`DELTA_SYNC` 或 `DIRECT_ACCESS` | `DELTA_SYNC` |
| `embedding_model_endpoint_name` |用于嵌入的 Databricks 服务端点 | `无` |
| `嵌入维度` |自我管理嵌入的维度 | `1536` |
| `端点类型` |端点类型（“STANDARD”或“STORAGE_OPTIMIZED”）| `标准` |
| `管道类型` |同步管道类型：`TRIGGERED` 或 `CONTINUOUS` | ‘触发’ |
| `仓库名称` | Databricks SQL 仓库名称（如果使用 SQL 仓库）| `无` |
| `查询类型` |查询类型：`ANN` 或 `HYBRID` | `安` |### 身份验证

Databricks 矢量搜索支持两种身份验证方法：

#### 服务主体（推荐用于生产）```python
config = {
    "vector_store": {
        "provider": "databricks",
        "config": {
            "workspace_url": "https://your-workspace.databricks.com",
            "client_id": "your-service-principal-id",
            "client_secret": "your-service-principal-secret",
            "endpoint_name": "your-endpoint",
            "catalog": "your_catalog",
            "schema": "your_schema",
            "table_name": "your_table",
            "collection_name": "your_index_name",
        }
    }
}
```
#### 个人访问令牌（用于开发）```python
config = {
    "vector_store": {
        "provider": "databricks",
        "config": {
            "workspace_url": "https://your-workspace.databricks.com",
            "access_token": "your-personal-access-token",
            "endpoint_name": "your-endpoint",
            "catalog": "your_catalog",
            "schema": "your_schema",
            "table_name": "your_table",
            "collection_name": "your_index_name",
        }
    }
}
```
### 嵌入选项

#### 自管理嵌入（默认）
使用您自己的嵌入模型并直接提供向量：```python
config = {
    "vector_store": {
        "provider": "databricks",
        "config": {
            # ... authentication config ...
            "embedding_dimension": 768,  # Match your embedding model
        }
    }
}
```
#### Databricks 计算嵌入
让 Databricks 使用服务端点计算文本的嵌入：```python
config = {
    "vector_store": {
        "provider": "databricks",
        "config": {
            # ... authentication config ...
            "embedding_model_endpoint_name": "e5-small-v2"
        }
    }
}
```
### 重要提示

- **索引类型**：此实现支持“DELTA_SYNC”（与源增量表自动同步）和“DIRECT_ACCESS”（直接管理向量）索引类型。
- **Unity Catalog**：源表和索引在指定的 `catalog.schema` 命名空间下创建。
- **端点自动创建**：如果指定的端点不存在，则会自动创建。
- **索引自动创建**：如果指定的索引不存在，它将使用提供的配置自动创建。
- **过滤器支持**：支持按元数据字段进行过滤，STANDARD 与 STORAGE_OPTIMIZED 端点具有不同的语法。
