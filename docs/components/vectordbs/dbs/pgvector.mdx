[pgvector](https://github.com/pgvector/pgvector) 是 Postgres 的开源向量相似性搜索扩展。连接到 Postgres 后，运行“CREATE EXTENSION IF NOT EXISTS vector;”来创建向量扩展。

### 用法

### 配置

以下是可用于配置 pgvector 的参数：

|参数|描述 |默认值 |
| ---| ---| ---|
| `数据库名称` |数据库名称 | `postgres` |
| `集合名称` |作品集名称| `mem0` |
| `embedding_model_dims` |嵌入模型的尺寸| `1536` |
| `用户` |连接数据库的用户名 | `无` |
| `密码` |连接数据库的密码 | `无` |
| `主机` |运行 Postgres 服务器的主机 | `无` |
| `端口` | Postgres 服务器运行的端口 | `无` |
| `diskann` |是否使用diskann进行向量相似度搜索（需要pgvectorscale） | '真实' |
| `hnsw` |是否使用hnsw进行向量相似度搜索 | ‘假’|
| `ssl 模式` |用于 PostgreSQL 连接的 SSL 模式（例如、“要求”、“首选”、“禁用”）| `无` |
| `连接字符串` | PostgreSQL 连接字符串（覆盖各个连接参数）| `无` |
| `连接池` | psycopg2 连接池对象（覆盖连接字符串和各个参数）| `无` |**注意（TypeScript OSS）：** 如果省略 `dbname`，TypeScript 客户端将使用数据库名称 `vector_store`。 Python 默认使用“postgres”作为“dbname”，如上表所示。

**注意**：连接参数具有以下优先级：
1. `connection_pool` (highest priority)
2.`连接字符串`
3. 单独的连接参数（`user`、`password`、`host`、`port`、`sslmode`）
