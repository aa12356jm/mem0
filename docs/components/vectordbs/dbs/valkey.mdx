# Valkey 矢量商店

[Valkey](https://valkey.io/) 是一个开源 (BSD) 高性能键/值数据存储，支持各种工作负载和丰富的数据结构，包括矢量搜索。

＃＃ 安装```bash
pip install mem0ai[vector_stores]
```
＃＃ 用法```python
config = {
    "vector_store": {
        "provider": "valkey",
        "config": {
            "collection_name": "test",
            "valkey_url": "valkey://localhost:6379",
            "embedding_model_dims": 1536,
            "index_type": "flat"
        }
    }
}

m = Memory.from_config(config)
messages = [
    {"role": "user", "content": "I'm planning to watch a movie tonight. Any recommendations?"},
    {"role": "assistant", "content": "How about thriller movies? They can be quite engaging."},
    {"role": "user", "content": "I'm not a big fan of thriller movies but I love sci-fi movies."},
    {"role": "assistant", "content": "Got it! I'll avoid thriller recommendations and suggest sci-fi movies in the future."}
]
m.add(messages, user_id="alice", metadata={"category": "movies"})
```
## 参数

以下是可用于配置 Valkey 的参数：

|参数|描述 |默认值 |
| ---| ---| ---|
| `集合名称` |存储向量的集合的名称 | `mem0` |
| `valkey_url` | Valkey 服务器的连接 URL | `valkey://localhost:6379` |
| `embedding_model_dims` |嵌入模型的尺寸| `1536` |
| `索引类型` |向量索引算法（`hnsw` 或 `flat`）| `hnsw` |
| `hnsw_m` | HNSW 的双向链路数量 | `16` |
| `hnsw_ef_construction` | HNSW 动态候选人名单的规模 | `200` |
| `hnsw_ef_runtime` |用于搜索的动态候选列表的大小 | `10` |
| `集群模式` |为 Valkey 集群 (CME) 部署启用集群模式 | `假` |
| `距离度量` |向量相似度的距离度量 | `余弦` |

## 集群模式

要在启用集群模式 (CME) 的情况下使用 Valkey，请将 `cluster_mode` 设置为 `true`：```python
config = {
    "vector_store": {
        "provider": "valkey",
        "config": {
            "collection_name": "memories",
            "valkey_url": "valkey://cluster-endpoint:6379",
            "embedding_model_dims": 1536,
            "cluster_mode": True
        }
    }
}
```
启用集群模式后，连接器使用“ValkeyCluster”而不是独立客户端，后者会自动处理“MOVED”/“ASK”重定向。 valkey-search 模块的内置协调器在所有分片之间协调搜索查询。有关集群模式行为的详细信息，请参阅 [valkey-search 文档](https://github.com/valkey-io/valkey-search)。
