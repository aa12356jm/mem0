[Upstash Vector](https://upstash.com/docs/vector) 是一个具有内置嵌入模型的无服务器矢量数据库。

### Upstash 嵌入的用法

您可以通过将“enable_embeddings”设置为“True”来启用内置嵌入模型。这允许您使用 Upstash 的嵌入模型进行矢量化。```python
import os
from mem0 import Memory

os.environ["UPSTASH_VECTOR_REST_URL"] = "..."
os.environ["UPSTASH_VECTOR_REST_TOKEN"] = "..."

config = {
    "vector_store": {
        "provider": "upstash_vector",
        "enable_embeddings": True,
    }
}

m = Memory.from_config(config)
m.add("Likes to play cricket on weekends", user_id="alice", metadata={"category": "hobbies"})
```
<注意>
    将“enable_embeddings”设置为“True”将绕过您配置的任何外部嵌入提供程序。
</注>

### 与外部嵌入提供程序一起使用```python
import os
from mem0 import Memory

os.environ["OPENAI_API_KEY"] = "..."
os.environ["UPSTASH_VECTOR_REST_URL"] = "..."
os.environ["UPSTASH_VECTOR_REST_TOKEN"] = "..."

config = {
    "vector_store": {
        "provider": "upstash_vector",
    },
    "embedder": {
        "provider": "openai",
        "config": {
            "model": "text-embedding-3-large"
        },
    }
}

m = Memory.from_config(config)
m.add("Likes to play cricket on weekends", user_id="alice", metadata={"category": "hobbies"})
```
### 配置

以下是可用于配置 Upstash Vector 的参数：

|参数|描述 |默认值 |
| ------------------- | ---------------------------------- | ------------- |
| `网址` | Upstash 向量索引的 URL | `无` |
| `令牌` | Upstash Vector 索引的代币 | `无` |
| `客户` |一个 `upstash_vector.Index` 实例 | `无` |
| `集合名称` |使用的默认命名空间 | `""` |
| `启用嵌入` |是否使用 Upstash 嵌入 | ‘假’|

<注意>
  当未提供“url”和“token”时，“UPSTASH_VECTOR_REST_URL”和
  使用“UPSTASH_VECTOR_REST_TOKEN”环境变量。
</注>
