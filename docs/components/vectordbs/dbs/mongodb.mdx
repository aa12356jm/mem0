# MongoDB

[MongoDB](https://www.mongodb.com/) 是一个多功能文档数据库，支持矢量搜索功能，允许对大型数据集进行高效的高维相似性搜索，并具有强大的可扩展性和性能。

＃＃ 用法```python
import os
from mem0 import Memory

os.environ["OPENAI_API_KEY"] = "sk-xx"

config = {
    "vector_store": {
        "provider": "mongodb",
        "config": {
            "db_name": "mem0-db",
            "collection_name": "mem0-collection",
            "mongo_uri":"mongodb://username:password@localhost:27017"
        }
    }
}

m = Memory.from_config(config)
messages = [
    {"role": "user", "content": "I'm planning to watch a movie tonight. Any recommendations?"},
    {"role": "assistant", "content": "How about thriller movies? They can be quite engaging."},
    {"role": "user", "content": "I’m not a big fan of thriller movies but I love sci-fi movies."},
    {"role": "assistant", "content": "Got it! I'll avoid thriller recommendations and suggest sci-fi movies in the future."}
]
m.add(messages, user_id="alice", metadata={"category": "movies"})
```
## 配置

以下是可用于配置 MongoDB 的参数：

|参数|描述 |默认值 |
| ---| ---| ---|
|数据库名称 | MongoDB 数据库名称 | `"mem0_db"` |
|集合名称 | MongoDB 集合的名称 | `"mem0_collection"` |
|嵌入_模型_尺寸 |嵌入向量的维度 | `1536` |
| mongo_uri | MongoDB URI 连接字符串 | `mongodb://用户名:密码@localhost:27017` |

> **注意**：如果未提供 `mongo_uri`，则默认为 `mongodb://username:password@localhost:27017`。
