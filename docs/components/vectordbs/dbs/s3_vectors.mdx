[Amazon S3 Vectors](https://aws.amazon.com/s3/features/vectors/) 是一种专门构建的、成本优化的矢量存储和查询服务，用于语义搜索和 AI 应用程序。它提供 S3 级别的弹性和持久性以及亚秒级查询性能。

### 安装

S3 Vectors 支持需要额外的依赖项。安装它们：```bash
pip install boto3
```
### 用法

要将 Amazon S3 Vectors 与 Mem0 结合使用，您需要拥有 AWS 账户和必要的 IAM 权限 (`s3vectors:*`)。确保您的环境配置了 AWS 凭证（例如，通过“~/.aws/credentials”或环境变量）。```python
import os
from mem0 import Memory

# Ensure your AWS credentials are configured in your environment
# e.g., by setting AWS_ACCESS_KEY_ID, AWS_SECRET_ACCESS_KEY, and AWS_DEFAULT_REGION

config = {
    "vector_store": {
        "provider": "s3_vectors",
        "config": {
            "vector_bucket_name": "my-mem0-vector-bucket",
            "collection_name": "my-memories-index",
            "embedding_model_dims": 1536,
            "distance_metric": "cosine",
            "region_name": "us-east-1"
        }
    }
}

m = Memory.from_config(config)
messages = [
    {"role": "user", "content": "I'm planning to watch a movie tonight. Any recommendations?"},
    {"role": "assistant", "content": "How about a thriller movie? They can be quite engaging."},
    {"role": "user", "content": "I'm not a big fan of thriller movies but I love sci-fi movies."},
    {"role": "assistant", "content": "Got it! I'll avoid thriller recommendations and suggest sci-fi movies in the future."}
]
m.add(messages, user_id="alice", metadata={"category": "movies"})
```
### 配置

以下是可用于配置 Amazon S3 矢量的参数：

|参数|描述 |默认值 |
| ---------------------- | -------------------------------------------------------------------------------- | -------------------------------------------------- |
| `向量桶名称` |要使用的 S3 矢量存储桶的名称。如果不存在，则会创建它。 |必填|
| `集合名称` |存储桶内向量索引的名称。                                  | `mem0` |
| `embedding_model_dims` |嵌入模型的尺寸。必须与您的嵌入器匹配。                     | `1536` |
| `距离度量` |用于相似性搜索的距离度量。选项：“余弦”、“欧氏”。           | `余弦` |
| `地区名称` |存储桶和索引所在的 AWS 区域。                                | “无”（使用 AWS 配置中的默认值）|

### IAM 权限您的 AWS 身份（用户或角色）需要权限才能对 S3 矢量执行操作。最低限度的政策如下所示：```json
{
  "Version": "2012-10-17",
  "Statement": [
    {
      "Effect": "Allow",
      "Action": "s3vectors:*",
      "Resource": "*"
    }
  ]
}
```
对于生产，建议将资源 ARN 的范围缩小到您的特定存储桶和索引。
