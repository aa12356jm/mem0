[Baidu VectorDB](https://cloud.baidu.com/doc/VDB/index.html)是百度智能云开发的企业级分布式矢量数据库服务。它由百度专有的“Mochow”矢量数据库内核提供支持，为矢量搜索提供高性能、可用​​性和安全性。

＃＃＃ 用法```python
import os
from mem0 import Memory

config = {
    "vector_store": {
        "provider": "baidu",
        "config": {
            "endpoint": "http://your-mochow-endpoint:8287",
            "account": "root",
            "api_key": "your-api-key",
            "database_name": "mem0",
            "table_name": "mem0_table",
            "embedding_model_dims": 1536,
            "metric_type": "COSINE"
        }
    }
}

m = Memory.from_config(config)
messages = [
    {"role": "user", "content": "I'm planning to watch a movie tonight. Any recommendations?"},
    {"role": "assistant", "content": "How about a thriller movie? They can be quite engaging."},
    {"role": "user", "content": "I'm not a big fan of thriller movies but I love sci-fi movies."},
    {"role": "assistant", "content": "Got it! I'll avoid thriller recommendations and suggest sci-fi movies in the future."}
]
m.add(messages, user_id="alice", metadata={"category": "movies"})
```
### 配置

以下是配置百度VectorDB可用的参数：

|参数|描述 |默认值 |
| ---| ---| ---|
| `端点` |百度 VectorDB 实例的端点 URL |必填 |
| `帐户` |百度VectorDB帐号名 | `根` |
| `api_key` |访问百度VectorDB的API密钥|必填 |
| `数据库名称` |数据库名称 | `mem0` |
| `表名` |表名 | `mem0_table` |
| `embedding_model_dims` |嵌入模型的尺寸| `1536` |
| `度量类型` |相似性搜索的距离度量 | 'L2' |

### 距离指标

支持以下距离度量：

- `L2`：欧氏距离（默认）
- `IP`：内积
- `COSINE`：余弦相似度

### 索引配置

矢量索引自动配置以下 HNSW 参数：

- `m`: 16（每个元素的连接数）
- `efconstruction`: 200（动态候选列表的大小）
- `auto_build`: true (自动构建索引)
- `auto_build_index_policy`：增量构建，增量为 10000 行
