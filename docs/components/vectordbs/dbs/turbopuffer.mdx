[Turbopuffer](https://turbopuffer.com) 是一个无服务器矢量数据库，针对大规模低延迟搜索进行了优化。它提供具有本机元数据过滤功能的经济高效的矢量存储。

＃＃＃ 用法```python
import os
from mem0 import Memory

os.environ["OPENAI_API_KEY"] = "sk-xx"
os.environ["TURBOPUFFER_API_KEY"] = "tpuf_xxxxxxxxxxxx"

config = {
    "vector_store": {
        "provider": "turbopuffer",
        "config": {
            "collection_name": "movie_preferences",
            "embedding_model_dims": 1536,
            "region": "gcp-us-central1",
        }
    }
}

m = Memory.from_config(config)

messages = [
    {"role": "user", "content": "I'm planning to watch a movie tonight. Any recommendations?"},
    {"role": "assistant", "content": "How about thriller movies? They can be quite engaging."},
    {"role": "user", "content": "I'm not a big fan of thrillers but I love sci-fi."},
    {"role": "assistant", "content": "Got it! I'll suggest sci-fi movies instead."}
]

m.add(messages, user_id="alice", metadata={"category": "movies"})

# Search memories
results = m.search(query="sci-fi recommendations", filters={"user_id": "alice"})
```
### 配置

以下是可用于配置 Turbopuffer 的参数：

|参数|描述 |默认值 |
| ---| ---| ---|
| `集合名称` |命名空间/集合的名称 | `mem0` |
| `embedding_model_dims` |嵌入模型的尺寸（必须与您选择的嵌入模型相匹配）| `1536` |
| `api_key` | Turbopuffer API 密钥 |环境变量：`TURBOPUFFER_API_KEY` |
| `地区` |涡轮喷气机区域 | `gcp-us-central1` |
| `距离度量` |向量相似度的距离度量（“cosine_distance”或“euclidean_squared”）| `余弦距离` |
| `batch_size` |批量操作的批量大小 | `100` |
| `extra_params` | Turbopuffer 客户端的附加参数 | `无` |

### 地区

|地区 |地点 |
| ---| ---|
| `gcp-us-central1` |美国爱荷华州（默认）|
| `aws-us-west-2` |美国俄勒冈州 |

### 配置示例```python
config = {
    "vector_store": {
        "provider": "turbopuffer",
        "config": {
            "collection_name": "my_memories",
            "embedding_model_dims": 1536,
            "api_key": "tpuf_xxxxxxxxxxxx",
            "region": "aws-us-west-2",
            "distance_metric": "cosine_distance",
            "batch_size": 200,
        }
    }
}
```
