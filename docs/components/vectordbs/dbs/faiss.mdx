[FAISS](https://github.com/facebookresearch/faiss) 是一个用于高效相似性搜索和密集向量聚类的库。它旨在处理大规模数据集，并为矢量数据提供高性能搜索引擎。 FAISS 针对内存使用和搜索速度进行了优化，使其成为生产环境的绝佳选择。

＃＃＃ 用法```python
import os
from mem0 import Memory

os.environ["OPENAI_API_KEY"] = "sk-xx"

config = {
    "vector_store": {
        "provider": "faiss",
        "config": {
            "collection_name": "test",
            "path": "/tmp/faiss_memories",
            "distance_strategy": "euclidean"
        }
    }
}

m = Memory.from_config(config)
messages = [
    {"role": "user", "content": "I'm planning to watch a movie tonight. Any recommendations?"},
    {"role": "assistant", "content": "How about thriller movies? They can be quite engaging."},
    {"role": "user", "content": "I'm not a big fan of thriller movies but I love sci-fi movies."},
    {"role": "assistant", "content": "Got it! I'll avoid thriller recommendations and suggest sci-fi movies in the future."}
]
m.add(messages, user_id="alice", metadata={"category": "movies"})
```
### 安装

要在 mem0 项目中使用 FAISS，您需要为您的环境安装适当的 FAISS 软件包：```bash
# For CPU version
pip install faiss-cpu

# For GPU version (requires CUDA)
pip install faiss-gpu
```
### 配置

以下是可用于配置 FAISS 的参数：

|参数|描述 |默认值 |
| ---| ---| ---|
| `集合名称` |作品集名称| `mem0` |
| `路径` | FAISS索引和元数据的存储路径| `/tmp/faiss/<集合名称>` |
| `距离策略` |使用的距离度量策略（选项：'euclidean'、'inner_product'、'cosine'）| `欧几里得` |
| `normalize_L2` |是否对L2向量进行归一化（仅适用于欧氏距离） | ‘假’|

### 性能考虑因素

FAISS 为矢量搜索提供了几个优势：

1. **效率**：FAISS针对内存使用和速度进行了优化，使其适合大规模应用。
2. **离线支持**：FAISS完全在本地工作，无需外部服务器或API调用。
3.**存储选项**：向量可以存储在内存中以获得最大速度或持久保存到磁盘。
4. **多种索引类型**：FAISS 支持针对各种用例优化的不同索引类型（尽管 mem0 目前使用基本平面索引）。

### 距离策略

mem0 中的 FAISS 支持三种距离策略：- **euclidean**：L2距离，适合大多数嵌入模型
- **inner_product**：点积相似度，对于一些专门的嵌入很有用
- **余弦**：余弦相似度，最适合比较语义相似度，无论向量大小如何

当将“cosine”或“inner_product”与归一化向量一起使用时，您可能需要设置“normalize_L2=True”以获得更好的结果。
